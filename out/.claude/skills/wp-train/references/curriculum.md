# WordPress 培训课纲

八层递进，从基础操作到独立排障。每层学完后有明确的能力里程碑。

## Layer 1: 初识 WordPress
**学完后能：设置好一个空白站点**

### 1.1 登录与仪表盘
- 登录后台、仪表盘导航、Admin Bar、站点健康
- 站点标题、副标题、管理员邮箱、成员注册开关

### 1.2 站点基础设置
- 固定链接、时区、日期/时间格式
- 讨论设置（评论开关）、阅读设置（每页文章数）

### 1.3 用户与个人资料
- 昵称/显示名称、创建用户、角色（管理员/编辑/作者/投稿者/订阅者）
- 默认角色、删除用户并归属内容

## Layer 2: 内容管理
**学完后能：给站点填充所有类型的内容**

### 2.1 页面管理
- 创建/发布页面、静态首页、子页面、slug 别名、页面可见性

### 2.2 文章与分类体系
- 分类（层级）、标签（扁平）、子分类、文章摘要、默认分类重命名

### 2.3 媒体管理
- 上传、Alt Text、图片尺寸设置、页面内插图、上传目录结构

### 2.4 区块编辑器基础
- Heading / Paragraph / List / Image / Columns / Separator / Buttons / Cover

### 2.5 菜单与导航
- 创建菜单、子菜单、自定义链接、多个菜单

### 2.6 外观基础
- 安装/切换主题、站点图标、Logo、全站编辑器模板编辑

### 2.7 插件管理
- 安装/激活/停用/删除、预装插件、批量操作

## Layer 3: 建站实战
**学完后能：独立交付一个客户项目**

### 3.1 SEO 基础
- 安装 SEO 插件、SEO 标题和 meta description
- slug 优化策略、H1/H2 标题层级规范、XML Sitemap

### 3.2 内容排版规范
- 标题层级（一个页面只有一个 H1）、段落长度、图文配合
- 列表和引用的恰当使用、CTA 布局

### 3.3 备份与恢复
- 导出/导入（Tools → Export/Import）
- 完整备份演练：导出 → 删除内容 → 导入恢复

### 3.4 响应式与预览
- 编辑器内移动端预览、响应式图片概念

### 3.5 客户建站项目
- 链式任务：从 brief 到完整交付
- 综合运用 L1-L3 所有技能
- Agent 协作：如何精确描述需求
