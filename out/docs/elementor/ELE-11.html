Design System Variables（字号/间距/圆角变量）

Elementor 建站L3NEW

📖 What — 这是什么

Elementor 的 Design System 允许你定义全局变量（Global Colors 和 Global Fonts），以及通过 Site Settings 统一管理字号、间距、圆角等设计参数。修改一处变量，所有引用该变量的元素同步更新。

🎯 Why — 为什么要做

  • 一致性：全站使用统一的字号/间距/颜色系统，避免「每个页面长得不一样」
  • 高效修改：改一个变量 = 改全站，不需要逐页调整
  • 团队协作：设计规范可视化，任何人编辑都能遵循同一套系统

🔧 How — 怎么做

全局颜色（Global Colors）

  1. 进入 Elementor 编辑器，点击左上角汉堡菜单 → Site Settings → Global Colors
  2. 定义核心色板：
    • Primary：主色调（品牌色、CTA 按钮）
    • Secondary：辅助色
    • Text：正文文字色
    • Accent：强调色（链接、hover 状态）
  3. 可点击 + 添加自定义颜色变量（如 Surface、Border）

全局字体（Global Fonts）

  1. 在 Site Settings 中进入 Global Fonts
  2. 定义 4 级字体：Primary / Secondary / Text / Accent
  3. 每个字体可设置：字体族、字重、大小、行高、字间距

使用变量

选中任意 Widget，在颜色选择器中点击地球图标可选择全局颜色；在字体设置中点击地球图标可选择全局字体。

自定义间距与圆角

Elementor 没有原生的全局间距/圆角变量，但可以通过以下方式实现：

  • CSS Variables：在 Site Settings → Custom CSS 中定义 :root { --spacing-sm: 8px; --radius: 12px; }，然后在各元素的 Custom CSS 中引用
  • Theme Style：在 Site Settings → Theme Style 中设置 Button 默认圆角、Form 字段样式等，统一基础组件
建立变量的投入在初期，回报在后期。3 个页面以上的项目就值得先花 10 分钟建立 Design System。

⚠️ 常见问题与注意事项

  • 解绑变量：点击颜色/字体选择器中的地球图标再点 X 可以断开全局变量引用，改为本地值
  • 默认变量 ≠ 自定义变量：Elementor 自带 4 个颜色和 4 个字体位置，修改这些会影响所有引用默认样式的元素
  • 导出限制：Global Colors/Fonts 随模板导出，但 Custom CSS 中的变量不会自动迁移到其他站点
