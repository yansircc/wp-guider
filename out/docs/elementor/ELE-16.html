Hero Section 设计

Elementor 建站L3

本文同时覆盖: ELE-16, ELE-17

📖 What — 这是什么

Hero Section 是页面最顶部的视觉焦点区域，通常占据首屏（Above the Fold），包含品牌核心信息、标题、副标题和行动按钮（CTA）。背景可以是静态图片、视频或渐变色。

一个好的 Hero Section 在 3 秒内告诉访客"你是谁、你做什么、接下来该做什么"。

🎯 Why — 为什么要做

  • 第一印象：Hero 是访客看到的第一个内容，直接决定他们是否继续浏览
  • 转化入口：CTA 按钮是引导用户进入下一步的关键
  • 品牌表达：通过视觉设计传递品牌调性（专业/活泼/科技感等）

🔧 How — 怎么做

图片背景 Hero（L3）

  1. 创建一个全宽 Container，设置 Min Height: 80vh（占屏幕高度 80%）
  2. 在 Style Tab → Background → Classic，上传背景图
  3. 设置 Position: Center Center，Size: Cover
  4. 添加 Background Overlay：颜色叠加层（如黑色 50% 透明度），确保文字可读
  5. 在 Container 内部放置：
    • Heading Widget（H1，白色，大字号）
    • Text Editor（副标题，白色或浅灰）
    • Button Widget（CTA 按钮，强调色背景）
  6. Container 对齐设为 Center / Center（水平垂直居中）

视频背景 Hero（L3）

  1. 同上创建全宽 Container
  2. Style Tab → Background → Video
  3. 输入 YouTube 链接或上传自托管 MP4
  4. 设置选项：
    • Start/End Time：截取视频片段
    • Play on Mobile：关闭（手机播放视频耗电耗流量）
    • Fallback Image：手机端或视频加载前显示的静态图
  5. 添加 Overlay 确保文字可读
  6. 内容结构与图片 Hero 相同

设计要点

  • 标题控制在 6-10 个字以内，简洁有力
  • CTA 按钮文案用行动词（"免费试用"、"获取报价"、"立即咨询"）
  • 文字与背景对比度要高，确保可读性
  • 响应式：手机端调小标题字号，按钮改为全宽

⚠️ 常见问题与注意事项

  • 图片尺寸：Hero 背景图建议 1920×1080px，压缩后控制在 200KB 以内
  • 视频文件：自托管视频控制在 5MB 以内，超过的用 YouTube 嵌入
  • Overlay 必须：没有叠加层的情况下，白色文字在浅色背景上几乎看不清
  • 移动端视频：建议关闭移动端视频自动播放，用 Fallback Image 替代
  • 100vh 陷阱：在手机上 100vh 可能包含浏览器地址栏高度，导致滚动体验不佳。建议用 80vh 或固定像素值
