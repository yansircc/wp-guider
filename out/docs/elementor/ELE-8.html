图标盒子 / 图片盒子 Widget

Elementor 建站L2NEW

📖 What — 这是什么

Icon Box（图标盒子）和 Image Box（图片盒子）是 Elementor 的组合型 Widget，将图标/图片 + 标题 + 描述文字封装在一个组件中。

  • Icon Box：图标 + 标题 + 描述，适合展示服务特性、功能优势
  • Image Box：图片 + 标题 + 描述，适合团队成员介绍、产品特点

相比手动用 Icon + Heading + Text 三个 Widget 拼凑，Box 类 Widget 结构更统一、样式更协调。

🎯 Why — 为什么要做

几乎每个商业网站都需要「服务特性」「为什么选择我们」「产品优势」这类内容区块。用 Icon Box / Image Box 可以快速构建整齐一致的卡片式布局，比手动拼接效率高且风格统一。

🔧 How — 怎么做

Icon Box 图标盒子

  1. 拖入 Icon Box Widget
  2. Content Tab：选择图标、输入标题和描述文字
  3. 设置 Position：Left（图标在左）/ Top（图标在上）/ Right
  4. Style Tab：分别设置图标、标题、描述的颜色和字号
  5. 图标的 Spacing 控制图标与文字的距离

Image Box 图片盒子

  1. 拖入 Image Box Widget
  2. 上传或选择图片，输入标题和描述
  3. 设置图片位置（Left / Top / Right）
  4. Style Tab：设置图片尺寸、圆角、阴影，以及文字样式

实战：3 列服务特性

  1. 创建一个 Row 方向的 Container
  2. 拖入 3 个 Icon Box，分别设置不同图标和文案
  3. 父 Container 设置 Gap 和 Align Items: Stretch（等高卡片）
  4. 每个 Icon Box 的 Advanced Tab 设置相同的 Padding 和背景色
先做好一个 Icon Box 的完整样式，然后右键「复制」再修改内容，保证风格一致。

⚠️ 常见问题与注意事项

  • Icon Box 和 Image Box 都支持链接——整个盒子可以变成可点击区域
  • Image Box 的图片建议统一尺寸（如都用 200×200 正方形），否则卡片高度不一致
  • 移动端建议将图标位置改为 Top（居中显示），Left 在小屏幕上可能布局拥挤
  • 如果需要更复杂的卡片布局（如带按钮、标签），考虑用嵌套 Container 手动构建
