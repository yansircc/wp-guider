Testimonial Carousel 评价轮播

Elementor 建站L2NEW

📖 What — 这是什么

Testimonial Carousel 是 Elementor Pro 的客户评价轮播 Widget，以卡片轮播形式展示多条客户评价/推荐语。每张卡片包含评价内容、客户姓名、头衔/公司、头像和星级评分。

轮播方式可以是自动播放、手动左右滑动或分页点导航，在有限的页面空间中展示大量社会证明（Social Proof）。

🎯 Why — 为什么要做

社会证明是影响购买决策的核心要素之一——92% 的消费者在购买前会查看评价。Testimonial Carousel 让你在首页或产品页高效展示客户好评，增强信任感。

轮播形式比静态列表更省空间，视觉上也更有动感，能吸引用户停留阅读。

🔧 How — 怎么做

1. 添加 Widget

从 Widget 面板搜索 Testimonial Carousel，拖入页面。

2. 添加评价内容

Content Tab → Slides：

  1. 点击每个 Slide 设置：评价内容、姓名、Title（职位/公司）、头像
  2. 点击 Add Item 添加更多评价卡片
  3. 建议至少 4-6 条评价，保证轮播有足够内容

3. 配置轮播行为

Content Tab → Additional Options：

  • Slides Per View：一屏显示几张卡片（桌面 2-3 张）
  • Slides to Scroll：每次滑动几张
  • Autoplay：是否自动播放（推荐开启，速度 3000-5000ms）
  • Pause on Hover：鼠标悬停时暂停（推荐开启）
  • Navigation：显示左右箭头和/或底部分页点

4. 样式美化

Style Tab 可设置卡片背景色、文字颜色、头像大小、间距以及导航箭头和分页点的样式。

评价内容要简短有力——每条控制在 2-3 句话。过长的评价用户不会读完，反而削弱效果。

⚠️ 常见问题与注意事项

  • 使用真实客户评价——虚假评价不仅不道德，用户也能分辨出来
  • 头像使用真实照片效果远好于占位图标——有头像的评价可信度更高
  • 移动端将 Slides Per View 调整为 1，保证卡片完整显示
  • 自动播放速度不宜过快——用户需要时间阅读内容，建议每条停留 4-5 秒
  • 如果评价数量少于一屏显示数，轮播效果会异常——确保评价数 > Slides Per View
