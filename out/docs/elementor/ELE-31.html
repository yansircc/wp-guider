表单集成（Mailchimp / HubSpot）

Elementor 建站L3NEW

📖 What — 这是什么

Elementor Pro 的 Form Widget 原生集成了多种营销和 CRM 平台，包括 Mailchimp（邮件营销）、HubSpot（CRM）、ActiveCampaign、ConvertKit 等。集成后，表单提交的数据会自动同步到这些平台，无需手动导入。

🎯 Why — 为什么要做

  • 自动化线索管理：询盘数据自动进入 CRM，销售团队即时跟进
  • 邮件营销自动化：新订阅者自动加入 Mailchimp 列表，触发欢迎邮件序列
  • 消除人工操作：不再需要手动从 WordPress 导出 CSV 再导入到营销平台

🔧 How — 怎么做

集成 Mailchimp

  1. WordPress 后台 → Elementor → Settings → Integrations
  2. 找到 Mailchimp 区域，填入 API Key（从 Mailchimp 账户 → Extras → API Keys 获取）
  3. 保存并验证连接
  4. 编辑表单 → Actions After Submit → 添加 Mailchimp 动作
  5. 选择目标 Audience（列表）和 Group（分组）
  6. 映射表单字段到 Mailchimp 字段（Email、First Name、Last Name 等）

集成 HubSpot

  1. Elementor → Settings → Integrations → HubSpot
  2. 点击 Connect 授权 HubSpot 账户
  3. 编辑表单 → Actions After Submit → 添加 HubSpot 动作
  4. 映射字段：将表单字段对应到 HubSpot 的 Contact Properties

⚠️ 常见问题与注意事项

  • 字段映射是关键：Email 字段必须映射，否则无法创建联系人记录
  • 双重确认（Double Opt-in）：Mailchimp 默认开启双重确认，用户提交后还需要在邮箱中点击确认链接才会入列表
  • API Key 权限：确保 API Key 有足够的读写权限
  • 测试提交：集成配置完后，用测试邮箱提交一次表单，确认数据正确同步到了对应平台
  • GDPR 合规：收集用户邮箱用于营销需要征得同意，建议在表单中加一个同意勾选框
