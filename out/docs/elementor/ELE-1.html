安装 Elementor Pro 并创建第一个页面

Elementor 建站L1

本文同时覆盖: ELE-1, ELE-2

📖 What — 这是什么

Elementor 是 WordPress 最流行的可视化页面构建器，采用前端拖拽方式编辑页面，所见即所得。Elementor Pro 是其付费版本，提供 Theme Builder、Form、Popup、WooCommerce 集成等高级功能。

安装 Elementor Pro 并创建第一个页面是学习 Elementor 建站的起点——你需要熟悉编辑器界面、Widget 面板、页面设置等基本概念。

🎯 Why — 为什么要做

  • 零代码建站：无需 HTML/CSS 知识即可创建专业页面
  • 实时预览：修改即时可见，大幅提升设计效率
  • Pro 功能：Theme Builder 让你控制整站布局（Header/Footer/模板），而不仅仅是页面内容
  • 行业标准：Elementor 是 WordPress 生态中使用最广泛的 Page Builder，市场占有率遥遥领先

🔧 How — 怎么做

第一步：安装 Elementor Pro

  1. 进入 插件 → 安装插件 → 上传插件
  2. 上传 elementor-pro-*.zip 文件并点击"立即安装"
  3. 安装完成后点击"启用插件"
  4. 进入 Elementor → License，输入授权码并激活
  5. 如果系统提示需要先安装免费版 Elementor，先到插件市场搜索安装 "Elementor" 免费版

第二步：创建第一个页面

  1. 进入 页面 → 新建页面
  2. 输入页面标题，如"首页"
  3. 点击蓝色按钮 "使用 Elementor 编辑"
  4. 进入编辑器后，认识界面布局：
    • 左侧面板：Widget 库（可搜索/拖拽到右侧画布）
    • 中间画布：页面实时预览区域
    • 底部工具栏：响应式切换、撤销/重做、发布按钮
    • 右键菜单：复制/粘贴/删除 Widget 或 Container
  5. 从左侧拖一个 Heading Widget 到画布，输入标题文字
  6. 点击左下角的 "发布" 按钮

编辑器核心操作

  • 拖拽：从左侧 Widget 面板拖到画布指定位置
  • 点选编辑：点击画布中的任意元素，左侧面板显示该元素的设置
  • 三个设置 Tab：Content（内容）/ Style（样式）/ Advanced（高级/边距/动效）
  • 右键操作：复制、粘贴样式、复制整个 Widget、删除

⚠️ 常见问题与注意事项

  • 白屏问题：安装后编辑器白屏，通常是 PHP 内存不足。在 wp-config.php 中添加 define('WP_MEMORY_LIMIT', '256M');
  • 授权限制：Elementor Pro 按站点数授权，一个授权码不能用在多个站点
  • 兼容性：部分主题会与 Elementor 冲突，推荐使用 Hello Elementor 主题（官方配套，最轻量）
  • 页面模板：在页面设置中选择 "Elementor Canvas"（全屏无 Header/Footer）或 "Elementor Full Width"（无侧边栏）
  • 保存习惯：编辑过程中定期点击"更新"，Elementor 也有自动保存和修订历史
