视频背景 Hero

Elementor 建站L3

📖 What — 这是什么

视频背景 Hero 是用视频替代静态图片作为首屏（Hero Section）背景的高级设计手法。视频自动循环播放、静音，上层叠加半透明遮罩和文字内容（标题 + CTA 按钮）。

这是高端品牌网站常见的设计模式——比静态图片更有视觉冲击力，能有效传达品牌调性和产品特点。

🎯 Why — 为什么要做

首屏是用户对网站的第一印象，决定了用户是否继续浏览。视频背景比静态图片更能抓住注意力——动态内容天然比静态内容更吸引眼球。

适合场景：品牌官网、旅游/酒店、活动推广、产品展示等需要视觉冲击的页面。

🔧 How — 怎么做

1. 创建 Hero Container

  1. 添加一个全宽 Container
  2. 设置 Min Height：100vh（全屏高度）或 80vh
  3. 设置内容垂直居中：Justify Content: Center、Align Items: Center

2. 设置视频背景

  1. 选中 Container → Style Tab → Background
  2. Background Type 选择 Video
  3. 视频来源：YouTube URL 或自托管 MP4 文件
  4. 设置选项：
    • 勾选 Loop（循环播放）
    • 勾选 Play on Mobile（可选，移动端流量考虑）
    • 设置 Fallback Image（视频加载前或移动端不播放时的备用图片）

3. 添加遮罩层

在 Background 设置中添加 Background Overlay：

  • 颜色：黑色，透明度 40-60%
  • 或使用渐变遮罩（底部更暗）增加文字可读性

4. 添加前景内容

在 Container 内拖入 Heading + Text Editor + Button，文字设为白色。

视频应 10-30 秒为佳，文件控制在 5-10MB 以内。过长或过大的视频会严重拖慢页面加载。

⚠️ 常见问题与注意事项

  • 性能：视频背景是性能杀手——务必压缩视频（推荐 HandBrake 工具），设置 Fallback Image
  • 移动端：部分移动浏览器不支持自动播放视频背景，Fallback Image 是必须的
  • 内容：视频内容应该是氛围性的（环境、品牌、抽象画面），不应包含需要阅读的文字或关键信息
  • 音频：背景视频必须静音——自动播放有声视频会让用户立即关闭页面
