Cloud Templates 跨站点复用

Elementor 建站L3NEW

📖 What — 这是什么

Elementor Cloud Templates（云模板）将你保存的模板同步到 Elementor 云端账户。任何使用同一 Elementor 账号的站点都可以访问这些云模板，无需手动导出/导入 JSON 文件。这是 Elementor Pro 的跨站协作功能。

🎯 Why — 为什么要做

  • 多站点效率：管理 5 个客户站点时，一个 Hero 模板保存到云端后所有站点都能用
  • 团队协作：设计师保存云模板后，建站人员在其他站点直接调用
  • 版本管理：云端模板独立于站点存在，不会因为站点重置而丢失

🔧 How — 怎么做

保存到云端

  1. 在 Elementor 编辑器中，保存模板时选择 Save to Cloud（而非仅本地）
  2. 或在 模板 → Saved Templates 中，将已有模板上传到云端
  3. 确保你已在 Elementor → Settings → Account 中登录了 Elementor 账号

在其他站点使用

  1. 在目标站点的 Elementor 编辑器中打开模板库（文件夹图标）
  2. 切换到 My Templates → Cloud 标签
  3. 找到需要的模板 → 点击 Insert
  4. 模板内容会插入到当前页面

⚠️ 常见问题与注意事项

  • 需要 Pro 授权：Cloud Templates 是 Elementor Pro 功能，所有使用的站点都需要有效授权
  • 图片仍然不跟随：和 JSON 导出一样，云模板中的图片引用原站 URL。插入到新站后检查图片是否正常显示
  • 存储限制：云端模板有数量/容量限制，具体取决于你的 Elementor 订阅计划
  • 网络依赖：如果 Elementor 云服务不可用（极少发生），云模板暂时无法访问。重要模板建议同时保存本地备份
