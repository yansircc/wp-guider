布局系统：Flexbox Container 与 Grid Container

Elementor 建站L2-L3

本文同时覆盖: ELE-3, ELE-4, ELE-5

📖 What — 这是什么

Container 是 Elementor 的核心布局单元，取代了旧版 Section/Column 系统。Elementor 提供两种 Container 类型：

  • Flexbox Container：基于 CSS Flexbox，擅长一维排列（水平或垂直），支持无限嵌套
  • Grid Container：基于 CSS Grid，擅长二维布局（同时控制行和列），适合矩阵式排列

两者互补：简单的行或列排列用 Flexbox，网格/矩阵用 Grid，也可以混合嵌套使用。

🎯 Why — 为什么要做

  • 更灵活的布局：Flexbox Container 比旧版 Section/Column 强大得多，可以轻松实现等高列、居中对齐、响应式换行
  • 更少的 DOM：Container 生成更简洁的 HTML，页面性能更好
  • 嵌套无限制：可以创建多层嵌套，实现任意复杂的布局
  • 二维控制：Grid 同时控制行和列，更适合卡片列表、图片画廊等矩阵式场景
  • 官方推荐：Elementor 已将 Container 作为默认布局方式，旧版 Section 将逐步淘汰

🔧 How — 怎么做

一、Flexbox Container

在编辑器画布中点击 "+" 按钮，选择 Container 结构（单列、两列、三列等预设），或从 Widget 面板拖入空 Container。

点选 Container 后，在 Layout 选项卡中设置：

  • Direction（方向）：Row（水平排列）/ Column（垂直堆叠）
  • Align Items（交叉轴对齐）：flex-start / center / flex-end / stretch
  • Justify Content（主轴分布）：start / center / space-between / space-around
  • Gap（间距）：子元素之间的统一间距
  • Wrap（换行）：No Wrap（不换行）/ Wrap（自动换行）

嵌套 Container 实操

  1. 在已有 Container 内部，拖入一个新的 Container Widget
  2. 子 Container 继承并可以覆盖父 Container 的样式
  3. 实际场景——两栏 + 左栏再分上下：
    • 创建一个 Row 方向的父 Container
    • 拖入两个子 Container（设置宽度比例 60% / 40%）
    • 在左侧子 Container 中，设为 Column 方向，再拖入标题和文本

宽度与尺寸

  • Content Width：Boxed（有最大宽度限制）/ Full Width（占满屏幕）
  • Width：可设为百分比、px、vw 等单位
  • Min Height：设置最小高度，常用于 Hero 区域（如 100vh）

二、Grid Container

  1. 在编辑器中点击 "+" 添加新元素，选择 Grid 类型的 Container
  2. 在 Layout 选项卡中设置：
    • Columns（列数）：设置网格列数（如 3 列）
    • Rows（行数）：通常设为 auto，让内容自动决定
    • Column Gap / Row Gap：列间距和行间距
  3. 将 Widget 拖入 Grid Container，它们会自动按网格排列
  4. 选中单个子元素，可以设置 Grid Span（跨越多列/多行），实现不规则布局

Grid 实用示例：服务卡片网格

  • 创建 3 列 Grid Container
  • 放入 6 个 Icon Box Widget，自动排成 2 行 × 3 列
  • 在平板断点改为 2 列，手机改为 1 列

⚠️ 常见问题与注意事项

  • Flexbox vs Grid 选择：单方向排列用 Flexbox，矩阵/网格用 Grid。两者可以混用（Grid 里嵌套 Flexbox Container）
  • 旧版迁移：如果站点使用旧版 Section/Column，可以在 Elementor → Settings → Features 中开启 Container，但需要逐页手动迁移
  • 嵌套层级：虽然理论上可以无限嵌套，但建议控制在 3-4 层以内，过深的嵌套会增加维护难度
  • Flexbox 速记：先记住两个最常用的组合：Row + space-between（水平等间距分布），Column + center（垂直居中堆叠）
  • Grid 实验性：Grid Container 在 Elementor 中相对较新，部分高级 Grid 特性（如命名区域）可能尚不支持
  • Navigator 面板：使用左下角的 Navigator（结构面板）来可视化查看嵌套层级，避免迷失在复杂布局中
