保存为模板 + 导入导出

Elementor 建站L2

📖 What — 这是什么

Elementor 允许你将设计好的页面或区块（Section/Container）保存为模板，以便在同站复用或导出为 .json 文件分享给其他站点。这是 Elementor 模板管理的基础功能。

🎯 Why — 为什么要做

  • 复用设计：一套设计好的 Hero 区块可以在多个页面中重复使用，不必每次重新搭建
  • 跨站迁移：将 A 站点的设计导出后导入到 B 站点，快速复制设计
  • 备份：导出模板文件作为设计备份，即使站点出问题也能恢复

🔧 How — 怎么做

保存整页为模板

  1. 在 Elementor 编辑器中，点击左下角的箭头 → Save as Template
  2. 给模板起个名字（如"首页模板 v2"）
  3. 保存后可以在 模板 → Saved Templates 中找到

保存区块为模板

  1. 右键点击某个 Container 或 Section
  2. 选择 Save as Template
  3. 命名后保存（如"Hero 区块 - 蓝色版"）

导出模板

  1. WordPress 后台 → 模板 → Saved Templates
  2. 找到目标模板 → 点击 Export
  3. 下载 .json 文件

导入模板

  1. 模板 → Saved Templates → Import
  2. 上传 .json 文件
  3. 导入后在编辑器中通过模板库插入到页面

⚠️ 常见问题与注意事项

  • 图片不随模板导出：模板中的图片引用的是原站 URL，导入到新站后需要重新上传或替换图片
  • 字体和颜色：如果原站设置了全局字体和颜色，导入站点也需要配置相同的全局设定，否则样式可能不一致
  • 插件依赖：如果模板中使用了第三方 Widget（如 JetElements），目标站点也必须安装该插件
  • 命名规范：模板多了以后很难找，建议用统一命名规则，如"类型-名称-版本"（Hero-蓝色-v2）
