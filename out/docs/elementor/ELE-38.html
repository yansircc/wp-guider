Single Post 文章模板

Elementor 建站L3

📖 What — 这是什么

Single Post 模板控制每篇文章的详情页布局。通过 Theme Builder 创建 Single Post 模板后，所有文章都会使用这个模板来显示，取代主题默认的文章详情页样式。

模板中使用 Dynamic Tags 动态拉取每篇文章的标题、内容、特色图、作者、日期等数据，所以只需设计一次，所有文章自动套用。

🎯 Why — 为什么要做

  • 统一文章样式：所有文章页面自动应用同一套设计，保持视觉一致
  • 完全自定义：主题默认文章页的布局通常不可调，Theme Builder 让你控制每个细节
  • 动态内容：模板自动拉取文章数据，无需逐篇编辑

🔧 How — 怎么做

  1. 进入 Templates → Theme Builder
  2. 点击 Single Post 区域的 "+"
  3. 选择预设模板或从空白开始
  4. 构建文章页布局，常用 Widget：
    • Post Title：动态显示文章标题
    • Featured Image：文章特色图
    • Post Content：文章正文内容
    • Post Info：作者、日期、分类、评论数
    • Author Box：作者头像和简介
    • Post Navigation：上一篇/下一篇导航
    • Related Posts：相关文章推荐
    • Post Comments：评论区
  5. 在编辑器底部工具栏点击 Preview Settings（眼睛图标），选择一篇具体文章预览效果
  6. 点击 Publish → Display Conditions 选择 All Singular → Posts

常见布局结构

典型的博客文章模板从上到下：特色图（全宽）→ 标题 + Meta 信息 → 正文内容 → 标签 → 作者简介 → 上下篇导航 → 相关文章 → 评论区。

⚠️ 常见问题与注意事项

  • Post Content Widget 是必需的 — 没有它文章正文不会显示
  • 预览时要切换不同文章查看效果，确保长短内容、有无特色图的文章都显示正常
  • Display Conditions 设为 "Posts" 只影响文章，不影响页面（Page）。如果也要控制页面模板，需要单独创建 Single Page 模板
  • 如果某些文章需要特殊布局，可以创建多个 Single Post 模板，通过 Display Conditions 指定不同分类使用不同模板
  • 文章页的侧边栏需要在模板中手动添加 Sidebar Widget，Theme Builder 模板默认不包含侧边栏
