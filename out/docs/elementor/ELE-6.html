基础与进阶 Widget

Elementor 建站L1-L2

本文同时覆盖: ELE-6, ELE-7, ELE-8, ELE-9

📖 What — 这是什么

Widget 是 Elementor 的内容构件，每个 Widget 负责呈现一种内容类型。拖拽到画布后，通过 Content / Style / Advanced 三个 Tab 来配置内容、外观和高级属性。

基础 Widget 包括标题、文本、图片、按钮、图标、间距等；进阶 Widget 包括图标盒子、图片盒子、视频嵌入和地图等。

🎯 Why — 为什么要做

  • 核心构件：这些 Widget 是搭建任何页面的基础，80% 的页面内容由它们组成
  • 统一操作模式：掌握这些 Widget 的操作方式后，使用其他 Widget 会非常快上手
  • 设计表达：组合使用这些基础 Widget，就能实现大多数常见的页面设计效果

🔧 How — 怎么做

基础 Widget（L1）

Heading 标题

  • 拖入画布后在 Content Tab 输入文字
  • 选择 HTML 标签（H1-H6）——H1 用于页面主标题，每页只用一次
  • Style Tab：字体、大小、颜色、对齐方式

Text Editor 文本编辑器

  • 富文本编辑器，支持段落、加粗、列表、链接等
  • 可切换到"文本"模式直接编写 HTML
  • Style Tab：排版、文字颜色、行高

Image 图片

  • 上传或从媒体库选择图片
  • 设置尺寸（Thumbnail / Medium / Large / Full / 自定义）
  • 添加链接和灯箱效果（点击放大）

Button 按钮

  • Content：文字、链接、图标
  • Style：类型（填充/描边/文字）、颜色、圆角、悬停效果
  • Size：XS / SM / MD / LG / XL

Icon 图标 / Spacer 间距 / Divider 分割线

  • Icon：从图标库选择，设置大小和颜色
  • Spacer：控制垂直间距，不同设备可设不同值
  • Divider：水平分割线，可调样式（实线/虚线/双线）和宽度

进阶 Widget（L2）

Icon Box 图标盒子 / Image Box 图片盒子

  • 将图标/图片与标题、描述组合为一个整体
  • 常用于服务介绍、特色展示区块
  • 支持设置图标/图片位置（左侧/顶部/右侧）

Video 视频

  • 支持 YouTube、Vimeo、Dailymotion 及自托管视频
  • 可设置封面图（Image Overlay）和播放图标
  • 自动播放、静音、循环等播放选项

Google Maps 谷歌地图

  • 输入地址自动定位
  • 设置缩放级别和地图高度
  • 注意：需要 Google Maps API Key（在 Elementor → Settings → Integrations 中配置）

⚠️ 常见问题与注意事项

  • 三 Tab 规律：所有 Widget 都遵循 Content → Style → Advanced 的设置结构，养成从左到右配置的习惯
  • 图片优化：上传前压缩图片（推荐 WebP 格式），大图会拖慢页面加载速度
  • 语义化标题：H1 用于页面主标题，H2 用于区块标题，H3 用于子标题，不要因为字号需求而乱用标题层级
  • 按钮链接：按钮的 Link 可以设为页面链接、锚点（#section-id）、电话（tel:）或邮件（mailto:）
  • 视频性能：嵌入视频会增加页面加载时间，建议使用封面图 + 点击播放模式
