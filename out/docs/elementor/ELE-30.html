表单提交后动作（邮件 / 重定向 / Webhook）

Elementor 建站L3NEW

📖 What — 这是什么

Elementor Form Widget 的 "Actions After Submit"（提交后动作）让你定义用户提交表单后自动执行的一系列操作。常见动作包括发送邮件通知、重定向到感谢页、发送 Webhook 到第三方系统等。一次提交可以触发多个动作。

🎯 Why — 为什么要做

  • 邮件通知：第一时间收到询盘通知，不漏客户
  • 感谢页重定向：给用户明确反馈 + 用于 Google Ads 转化追踪
  • Webhook：将表单数据推送到 CRM（如 HubSpot）、消息系统（如 Slack）或自定义 API

🔧 How — 怎么做

  1. 选中页面上的 Form Widget
  2. 在 Content Tab 找到 Actions After Submit
  3. 点击 + Add Action 添加动作

动作一：Email（邮件通知）

  • To：收件人邮箱（如 sales@example.com）
  • Subject：邮件主题（支持动态字段，如 [field id="name"] 的询盘）
  • Message：邮件正文（使用 [all-fields] 输出所有字段内容）
  • From：发件人邮箱和名称

动作二：Redirect（重定向）

  • 填入感谢页 URL（如 /thank-you/）
  • 提交成功后自动跳转

动作三：Webhook

  • 填入 Webhook URL（接收端提供的 API 地址）
  • 表单数据会以 JSON 格式 POST 到该 URL
  • 可在 Advanced 中自定义字段映射

⚠️ 常见问题与注意事项

  • 动作执行顺序：动作按列表顺序依次执行。建议先 Email、再 Webhook、最后 Redirect
  • 邮件不稳定：Elementor 邮件也依赖 wp_mail()，同样建议安装 SMTP 插件
  • Redirect 要放最后：重定向后用户离开当前页面，如果在 Redirect 之后还有其他动作可能不会执行
  • Webhook 调试：先用 webhook.site 或 requestbin.com 测试，确认数据格式正确后再接入正式系统
  • 可添加多个 Email：如果需要通知多人，可以添加多个 Email 动作
