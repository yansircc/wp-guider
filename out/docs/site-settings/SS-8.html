日期和时间格式

站点设置 L1

📖 What — 这是什么

WordPress 允许自定义日期和时间的显示格式。这些格式会应用于前台文章发布日期、评论时间、归档页等所有时间显示的位置。

格式使用 PHP 的日期格式字符，如 Y-m-d 表示 2024-01-15，F j, Y 表示 January 15, 2024。

🎯 Why — 为什么要做

不同地区对日期格式有不同的习惯。中文站通常使用 Y年n月j日，英文站用 F j, Y，而 ISO 格式 Y-m-d 适合国际化站点。

选择合适的日期格式能提升用户体验，让访客一眼看懂发布时间。

🔧 How — 怎么做

  1. 登录 WordPress 后台
  2. 进入 设置 → 常规（Settings → General）
  3. 找到 日期格式（Date Format）区域，选择预设格式或输入自定义格式
  4. 找到 时间格式（Time Format）区域，选择 12 小时制或 24 小时制
  5. 点击 保存更改

常用格式参考

  • Y-m-d → 2024-01-15（推荐，国际通用）
  • Y年n月j日 → 2024年1月15日（中文站）
  • F j, Y → January 15, 2024（英文站）
  • H:i → 14:30（24小时制）
  • g:i a → 2:30 pm（12小时制）

⚠️ 常见问题与注意事项

  • 某些主题有自己的日期格式设置，可能会覆盖这里的全局设置
  • 格式字符区分大小写：m 是月份（01-12），M 是月份缩写（Jan-Dec）
  • 修改格式只影响显示，不影响数据库中存储的时间数据
