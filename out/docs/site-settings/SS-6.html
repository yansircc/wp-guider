修改固定链接

站点设置 L1

📖 What — 这是什么

固定链接（Permalinks）是 WordPress 页面和文章的 URL 结构。默认情况下 WordPress 使用 ?p=123 这样的丑陋链接格式。通过修改固定链接结构，可以让 URL 变成可读的形式，如 /sample-post/。

固定链接设置决定了整个站点的 URL 规则，是建站后必须首先配置的选项之一。

🎯 Why — 为什么要做

搜索引擎偏好包含关键词的、可读的 URL。/product-led-lighting/ 比 /?p=42 对 SEO 更友好，对用户也更直观。

固定链接必须在建站初期就设定好。如果网站已经运营一段时间后再修改，所有旧的 URL 都会失效（404），需要额外配置 301 重定向，非常麻烦。

🔧 How — 怎么做

  1. 登录 WordPress 后台
  2. 进入 设置 → 固定链接（Settings → Permalinks）
  3. 选择 文章名（Post name）结构，即 /%postname%/
  4. 点击 保存更改

推荐使用 "文章名" 结构，这是最简洁且 SEO 最友好的格式。不建议使用包含日期的结构，除非是新闻类网站。

⚠️ 常见问题与注意事项

  • 时机很关键：一定要在发布内容之前设置好固定链接，避免后期更改造成链接断裂
  • 如果保存后前台页面出现 404，通常是服务器的 .htaccess 文件没有正确写入，检查文件权限
  • Nginx 服务器不使用 .htaccess，需要在 Nginx 配置中添加 rewrite 规则
  • 自定义结构中不要使用中文，slug 应使用英文或拼音
