Yoast：设置页面 SEO 标题与描述

SEO L2

📖 What — 这是什么

SEO 标题（Title Tag）和描述（Meta Description）是搜索引擎结果页（SERP）展示的核心内容。Yoast 让你在每个页面/文章的编辑界面单独设置，覆盖默认生成的标题。

🔧 How — 怎么做

  1. 编辑任意页面或文章
  2. 滚动到编辑器下方的 Yoast SEO 面板
  3. 点击「SEO 标题」输入框：
    • 删除默认变量（如 %%title%% %%sep%% %%sitename%%）
    • 输入自定义标题，建议 50-60 个字符以内
  4. 点击「元描述」输入框：
    • 输入页面摘要，建议 120-155 个字符
    • 包含核心关键词，用自然语言描述页面内容
  5. 观察上方的搜索结果预览，确认展示效果
  6. 保存/更新页面
Yoast 用绿色/橙色/红色进度条提示标题和描述的长度是否合适。绿色 = 长度合适。

⚠️ 常见问题与注意事项

  • 变量可混用：可以在自定义标题中混用变量，如 我的标题 %%sep%% %%sitename%%
  • 描述不保证展示：Google 可能根据搜索词自动截取页面内容作为描述，但写好 meta description 仍然重要
  • 首页标题：进入 Yoast SEO → 搜索外观 → 常规 修改首页的 SEO 标题和描述
  • 不要堆关键词：标题和描述应自然通顺，搜索引擎会惩罚关键词堆砌
