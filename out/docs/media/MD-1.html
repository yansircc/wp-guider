上传图片与编辑图片信息

媒体管理 L1

📖 What — 这是什么

媒体库（Media Library）是 WordPress 集中管理所有上传文件的地方，包括图片、视频、PDF、音频等。上传图片到媒体库是向网站添加视觉内容的第一步。

上传后的图片存储在服务器的 wp-content/uploads/ 目录中，按年月自动分类。WordPress 会为每张图片自动生成多种尺寸的缩略图。

每张图片都有一组元数据可以编辑，其中最重要的是替代文本（Alt Text）——图片无法加载时显示的描述，也是屏幕阅读器和搜索引擎理解图片内容的唯一依据。

🎯 Why — 为什么要做

图片是网页内容的重要组成部分。产品展示、文章配图、Banner 横幅都需要通过媒体库上传。通过媒体库管理图片，可以在网站任何位置重复使用同一张图片。

Alt Text 的重要性常被忽视。视障用户依靠屏幕阅读器浏览网页，Alt Text 是他们「看到」图片的唯一方式。搜索引擎也依赖 Alt Text 理解图片含义，有助于图片出现在搜索结果中。

🔧 How — 怎么做

上传图片

  1. 进入后台 媒体 → 添加新文件
  2. 将图片文件拖拽到上传区域，或点击 选择文件 浏览本地文件
  3. 支持批量上传：可以同时选择多个文件

也可以在编辑文章时，点击 + 添加「图片」区块，选择 上传 直接添加。

编辑图片信息

  1. 进入 媒体 → 媒体库，点击目标图片
  2. 在附件详情面板中填写：
    • 替代文本：简洁描述图片内容，如「团队成员在会议室讨论方案」
    • 标题：图片标题，鼠标悬停时可能显示
    • 说明：图片下方的可见说明文字
    • 描述：可选，更详细的描述
  3. 修改会自动保存

⚠️ 常见问题与注意事项

  • 文件大小限制：默认取决于服务器配置（通常 2MB-128MB）。上传失败时检查 php.ini 的 upload_max_filesize
  • 支持的格式：默认支持 JPG、PNG、GIF、WebP。SVG 需要额外插件
  • 文件命名：上传前改为英文描述性名称（如 homepage-banner.jpg），避免中文或特殊字符
  • Alt Text 不要堆砌关键词：应自然描述图片内容。装饰性图片（分隔线等）的 Alt Text 可留空
  • 媒体库 vs 编辑器修改：媒体库中修改 Alt Text 更新全局默认值；编辑器中修改只影响当前文章中该图片实例
  • 图片尺寸：上传前建议压缩到合适尺寸，避免 5000x3000 像素的原始照片浪费带宽
