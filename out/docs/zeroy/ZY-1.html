安装 ZeroY 并配置 API Token

ZeroY AI 建站L1

本文同时覆盖: ZY-1, ZY-2

📖 What — 这是什么

ZeroY 是一款 AI 驱动的 WordPress 主题 + 可视化编辑器插件。安装后，你可以用自然语言描述需求，AI 帮你生成完整的网页布局和内容。API Token 是 ZeroY 云端服务的身份凭证，必须正确配置才能使用编辑器功能。

🎯 Why — 为什么要做

ZeroY 是整个建站流程的基础工具。主题提供全站框架，编辑器插件提供可视化操作能力，API Token 则打通本地 WordPress 与 ZeroY 云端 AI 服务的通道。没有正确安装和配置，后续所有页面编辑、模板创建、AI 生成功能都无法使用。

🔧 How — 怎么做

第一步：下载安装包

  1. 访问 zeroy.app，登录账户
  2. 在控制台下载 ZeroY 主题文件（.zip）和编辑器插件文件（.zip）

第二步：安装主题

  1. 进入 WordPress 后台 → 外观 → 主题 → 安装主题 → 上传主题
  2. 选择下载的主题 .zip 文件 → 安装 → 启用

第三步：安装编辑器插件

  1. 进入 WordPress 后台 → 插件 → 安装插件 → 上传插件
  2. 选择下载的编辑器插件 .zip 文件 → 安装 → 启用

第四步：绑定域名（关键前置步骤）

  1. 回到 zeroy.app 控制台
  2. 找到域名绑定设置，添加你的 WordPress 站点域名
  3. 确认绑定成功后再进行下一步

第五步：配置 API Token

  1. 在 zeroy.app 控制台复制你的 API Token
  2. 回到 WordPress 后台 → ZeroY 模块
  3. 将 API Token 粘贴到对应输入框 → 保存

⚠️ 常见问题与注意事项

  • 必须先绑定域名再填 Token：如果域名未绑定，Token 验证会失败，编辑器无法连接云端服务
  • 主题和插件都要装：只装主题没有编辑器功能，只装插件没有主题框架，两者缺一不可
  • 注意版本匹配：主题和插件版本需要匹配，建议同时从 zeroy.app 下载最新版
  • Token 泄露风险：API Token 等同于账户凭证，不要分享给他人或提交到代码仓库
