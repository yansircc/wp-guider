安装 SMTP 插件解决邮件不稳定

ZeroY AI 建站L3NEW

📖 What — 这是什么

WordPress 默认用 PHP wp_mail() 函数发送邮件，依赖服务器的 sendmail 服务。很多主机（尤其是共享主机）的 sendmail 不可靠，邮件经常进垃圾箱或根本发不出去。SMTP 插件让 WordPress 通过专业邮件服务商发信，从根本上解决邮件不稳定问题。

🎯 Why — 为什么要做

ZeroY 表单提交后的邮件通知如果收不到，你就无法及时跟进客户询盘。这是外贸网站的致命问题——客户提交了询盘但你不知道，白白丢失商机。SMTP 是解决这个问题的行业标准方案。

🔧 How — 怎么做

第一步：安装 Fluent SMTP 插件

  1. WordPress 后台 → 插件 → 安装插件
  2. 搜索 FluentSMTP
  3. 安装 → 启用

第二步：配置 SMTP 连接

  1. 进入 设置 → FluentSMTP
  2. 选择邮件服务商（支持 Gmail、Outlook、Amazon SES、自定义 SMTP 等）
  3. 填写发件人邮箱（From Email）和发件人名称
  4. 根据选择的服务商填入 SMTP 服务器地址、端口、用户名、密码或 API Key
  5. 保存连接设置

第三步：发送测试邮件

  1. 在 FluentSMTP 设置页找到发送测试邮件
  2. 填入你的收件邮箱 → 发送
  3. 检查是否收到测试邮件（也检查垃圾箱）

⚠️ 常见问题与注意事项

  • 推荐 FluentSMTP：免费、轻量、支持多种服务商，是目前最受欢迎的 SMTP 插件之一
  • Gmail 有每日发送限额：免费 Gmail 每天 500 封，Google Workspace 每天 2000 封。外贸网站建议用专业邮件服务
  • 发件人邮箱要和域名匹配：用 info@yourdomain.com 而非个人 Gmail 作为发件人，否则更容易进垃圾箱
  • SPF/DKIM 记录：在域名 DNS 中配置 SPF 和 DKIM 记录，进一步提升邮件送达率
  • 邮件日志：FluentSMTP 自带发送日志，可以在后台查看每封邮件的发送状态，方便排障
