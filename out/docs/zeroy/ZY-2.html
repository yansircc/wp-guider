配置 API Token（zeroy.app 控制台绑定域名）

ZeroY AI 建站L1NEW

📖 What — 这是什么

API Token 是 ZeroY 云端服务的身份凭证。你需要在 zeroy.app 控制台绑定网站域名，获取 Token，然后在 WordPress 后台填入，才能打通本地 WordPress 与 ZeroY 编辑器云端功能的通道。

🎯 Why — 为什么要做

没有正确配置 Token，ZeroY 编辑器无法连接云端服务，所有页面编辑、AI 生成、模板创建功能都会失败。这是安装 ZeroY 后必须完成的第一步配置。

🔧 How — 怎么做

第一步：绑定域名（关键前置步骤）

  1. 访问 zeroy.app，登录你的账户
  2. 进入控制台，找到域名绑定设置
  3. 添加你的 WordPress 站点域名（如 example.com）
  4. 确认绑定状态显示为成功
必须先绑定域名再填 Token。如果域名未绑定，Token 验证会失败。

第二步：复制 API Token

  1. 在 zeroy.app 控制台找到 API Token 区域
  2. 点击复制按钮，完整复制 Token 字符串

第三步：填入 WordPress 后台

  1. 回到 WordPress 后台 → ZeroY 模块
  2. 将 API Token 粘贴到对应输入框
  3. 点击保存
  4. 页面应提示连接成功

⚠️ 常见问题与注意事项

  • Token 泄露风险：API Token 等同于账户凭证，不要分享或提交到代码仓库
  • 域名要完全匹配：绑定时填写的域名必须和 WordPress 站点地址一致，注意 www 与非 www 的区别
  • 多站点场景：每个站点需要单独绑定域名、单独配置 Token
  • Token 失效：如果之后突然无法使用编辑器，回 zeroy.app 检查 Token 是否过期或被重置
