配置感谢页跳转（表单提交 → 设置 → 感谢页 URL）

ZeroY AI 建站L2NEW

📖 What — 这是什么

当用户通过 ZeroY 表单提交询盘后，可以自动跳转到一个「感谢页」（Thank You Page）。你需要先创建感谢页面，然后在表单设置中配置跳转 URL。

🎯 Why — 为什么要做

感谢页有两大作用：一是给用户明确的「提交成功」反馈，提升体验；二是作为 Google Ads 转化追踪的触发页面——用户到达感谢页 = 完成一次转化。没有感谢页，就无法精确追踪广告效果。

🔧 How — 怎么做

第一步：创建感谢页

  1. WordPress 后台 → 页面 → 新增
  2. 标题填 "Thank You" 或 "感谢您的咨询"
  3. 用 ZeroY 编辑器设计页面内容（感谢文字、下一步引导、返回首页按钮等）
  4. 发布页面，记下页面 URL（如 /thank-you/）

第二步：配置表单跳转

  1. 进入需要配置的表单所在页面，用 ZeroY 编辑器打开
  2. 选中表单元素 → 打开设置面板
  3. 找到「感谢页 URL」或「提交后跳转」选项
  4. 填入感谢页的完整路径（如 /thank-you/）
  5. 保存
ZeroY 表单的提交端点是 /wp-json/zeroy/v1/submissions，确保表单 action 指向此路径。

⚠️ 常见问题与注意事项

  • 感谢页必须已发布：草稿状态的页面无法访问，用户会看到 404
  • URL 要准确：建议用相对路径（/thank-you/）而非完整域名，避免开发/生产环境域名不同导致跳转失败
  • 表单不跳转？：首先检查固定链接设置（必须是「文章名」格式），再检查表单路径是否正确。详细排障见 ZY-38
  • 不要用感谢页做 SEO：感谢页应该加 noindex 标签，防止被搜索引擎收录
