修改页面作者

页面管理 L2

📖 What — 这是什么

WordPress 每个页面都有一个"作者"属性，记录是谁创建了这个页面。修改页面作者是指将页面的归属从一个用户转移到另一个用户，这个操作不会改变页面内容。

🎯 Why — 为什么要做

常见场景：员工离职后需要将其创建的页面归属转移；管理员用临时账号创建的内容需要归到正式账号；多人协作时需要重新分配内容所有权。

某些主题会在页面上显示作者名，修改作者可以确保显示信息准确。

🔧 How — 怎么做

方法一：通过页面编辑器（古腾堡）

  1. 进入 页面 → 所有页面，找到目标页面并点击编辑
  2. 在右侧设置面板，点击 页面 标签
  3. 找到 作者 下拉菜单（如果没看到，点击面板顶部的"页面"标签展开）
  4. 从下拉列表中选择新作者
  5. 点击 更新 保存

方法二：通过快速编辑

  1. 进入 页面 → 所有页面
  2. 将鼠标悬停在目标页面标题上，点击 快速编辑
  3. 在展开的编辑面板中找到 作者 下拉菜单
  4. 选择新作者，点击 更新

⚠️ 常见问题与注意事项

  • 只有管理员和编辑角色可以修改页面作者，作者和投稿者角色无此权限
  • 下拉菜单中只会显示有权发布页面的用户（管理员、编辑、作者）
  • 修改作者不会影响页面的创建时间和修改时间
  • 如果需要批量修改多个页面的作者，可以在页面列表中勾选多个页面，使用"批量操作 → 编辑"一次性修改
  • 删除用户时，WordPress 会提示将其内容归属到其他用户，不要选择"删除所有内容"
