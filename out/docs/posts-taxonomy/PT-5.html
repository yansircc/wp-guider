文章+标签+摘要

文章与分类 L2

📖 What — 这是什么

这个操作将文章发布的三个关键元素结合在一起：内容编写、标签标注和摘要设置。摘要（Excerpt）是文章的简短描述，通常在文章列表页、搜索结果和 RSS 订阅中显示。

WordPress 支持两种摘要：手动编写的自定义摘要，和从文章开头自动截取的自动摘要（默认 55 个词）。手动摘要的优先级高于自动摘要。

🎯 Why — 为什么要做

好的摘要决定读者是否点击阅读全文。自动截取的摘要往往在句子中间断开，生硬且不吸引人。手动编写的摘要可以精准提炼文章核心观点，大幅提升点击率。

搜索引擎也可能使用摘要作为搜索结果的描述（meta description），一段精心编写的摘要同时服务于用户体验和 SEO。

🔧 How — 怎么做

  1. 进入 文章 → 添加文章，输入标题和正文内容
  2. 在右侧面板的 标签 区域，输入相关标签并回车确认（3-5 个为宜）
  3. 在右侧面板向下滚动，找到 摘要（Excerpt）区域
  4. 如果没有看到摘要区域：
    • 点击右上角三点菜单 → 偏好设置 → 面板
    • 确保「摘要」选项已开启
  5. 在摘要输入框中编写 1-2 句话的文章概要（50-160 字为佳）
  6. 确认分类、标签、摘要都设置完毕后，点击 发布

⚠️ 常见问题与注意事项

  • 摘要面板默认可能隐藏：区块编辑器中摘要面板需要手动开启，很多新手找不到这个选项
  • 摘要长度建议：中文 50-80 字、英文 120-160 字符。过长的摘要在列表页会被截断
  • 摘要不支持 HTML：摘要字段是纯文本，不能添加加粗、链接等格式
  • 摘要与 SEO 插件的关系：如果安装了 Yoast SEO 或 Rank Math，它们有独立的 meta description 字段，优先级高于 WordPress 原生摘要
  • 不要在摘要中重复标题：摘要应补充标题未传达的信息，而非重复
