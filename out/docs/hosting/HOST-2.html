hPanel 控制面板导航

主机 L1

📖 What — 这是什么

hPanel 是 Hostinger 自研的主机控制面板，替代了传统的 cPanel。通过 hPanel 可以管理网站、域名、文件、数据库、邮箱、SSL 证书、备份等所有主机功能。

登录 Hostinger 账号后即可进入 hPanel，每个网站有独立的管理页面。

🎯 Why — 为什么要做

熟悉 hPanel 是使用 Hostinger 的基础。教练需要知道各功能区在哪里，才能快速完成主机层面的配置操作（如 SSL 开启、PHP 版本切换、文件管理等）。

🔧 How — 怎么做

登录 Hostinger 后，选择一个网站进入 hPanel，主要功能区域包括：

常用功能模块

  • Dashboard — 网站概览、快速操作入口
  • Websites — 管理所有网站列表、一键安装 WordPress
  • Domains — 域名管理、DNS 设置、域名转入
  • Files → File Manager — 在线文件管理器，可直接编辑服务器文件
  • Databases — MySQL 数据库管理、phpMyAdmin 入口
  • Emails — 企业邮箱配置
  • SSL — SSL 证书管理、强制 HTTPS
  • Advanced → PHP Configuration — PHP 版本和参数设置
  • Advanced → SSH Access — SSH 连接信息
  • Backups — 自动备份和恢复

快速找到功能

hPanel 顶部有搜索框，直接输入关键词（如 "SSL"、"PHP"、"backup"）可以快速定位功能页面。

⚠️ 常见问题与注意事项

  • hPanel 界面会不定期更新，功能位置可能微调，以实际页面为准
  • 注意区分"账户级别"和"网站级别"的设置 — 有些设置是针对整个 Hostinger 账户的，有些是针对单个网站的
  • 如果找不到某个功能，可能是当前方案不支持（如免费方案不含 SSH）
  • hPanel 支持中文界面，可在右上角切换语言
