创建编辑用户

用户管理 L1

📖 What — 这是什么

WordPress 支持多用户协作。管理员可以在后台创建新用户并分配角色。"编辑"（Editor）是 WordPress 的内置角色之一，拥有管理所有文章和页面的权限，但无法修改站点设置或安装插件。

创建编辑用户是团队协作的基础操作。

🎯 Why — 为什么要做

建站项目中通常需要为客户或内容编辑人员创建专属账号。给内容编辑人员分配"编辑"角色，既能让他们管理文章内容，又不会误操作站点核心设置。

这是最小权限原则的实践：给每个人刚好够用的权限，不多不少。

🔧 How — 怎么做

  1. 登录 WordPress 后台（需管理员权限）
  2. 进入 用户 → 添加用户（Users → Add New User）
  3. 填写必填字段：
    • 用户名（Username）— 登录名，创建后不可修改
    • 电子邮件（Email）— 必须唯一
  4. 设置密码：点击 显示密码 可自定义密码，或使用自动生成的强密码
  5. 在 角色（Role）下拉菜单中选择 编辑（Editor）
  6. 点击 添加用户

⚠️ 常见问题与注意事项

  • 用户名一旦创建就无法修改，建议使用规范的命名（如 firstname.lastname）
  • 邮箱地址必须唯一，不能与已有用户重复
  • 如果勾选了"发送用户通知"，新用户会收到包含密码设置链接的邮件
  • 编辑角色可以修改和删除所有人的文章，但不能安装插件或修改主题
