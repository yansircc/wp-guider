创建作者用户

用户管理 L1

📖 What — 这是什么

"作者"（Author）是 WordPress 内置角色之一。作者可以撰写和发布自己的文章、上传媒体文件、编辑和删除自己的已发布文章，但不能修改别人的文章。

创建作者用户的流程与创建编辑用户类似，只是在角色选择时选"作者"。

🎯 Why — 为什么要做

当客户需要多人撰写内容但互不干扰时，作者角色是最合适的选择。每个作者只能管理自己的文章，不会误操作他人的内容。

与编辑角色的区别：编辑可以管理所有人的文章，作者只能管理自己的。根据实际需求选择角色。

🔧 How — 怎么做

  1. 登录 WordPress 后台（需管理员权限）
  2. 进入 用户 → 添加用户（Users → Add New User）
  3. 填写用户名和邮箱
  4. 设置密码
  5. 在 角色（Role）下拉菜单中选择 作者（Author）
  6. 点击 添加用户

⚠️ 常见问题与注意事项

  • 作者可以删除自己已发布的文章，如果不希望作者删除内容，可以改用"投稿者"角色
  • 作者可以上传文件到媒体库，投稿者不能
  • 作者发布的文章，WordPress 会自动生成该作者的归档页（/author/username/）
  • 如果需要更精细的权限控制，可以使用 User Role Editor 插件自定义角色权限
