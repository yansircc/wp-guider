在 Cloudflare 注册域名

域名 L1

📖 What — 这是什么

Cloudflare Registrar 提供域名注册服务，最大特点是零加价 — 域名以批发价出售，没有任何利润加成。.com 域名年费约 $10，且续费价格同样是批发价。

在 Cloudflare 注册的域名自动接入 Cloudflare DNS 和 CDN 服务，无需额外配置 NS 记录。

🎯 Why — 为什么要做

如果你已经在使用 Cloudflare 的 DNS 或 CDN 服务（大多数站点都会用），直接在 Cloudflare 注册域名是最省事的方案。省去了修改 NS 记录的步骤，管理也集中在一个平台。

Cloudflare 的域名自动包含免费的 WHOIS 隐私保护和 DNSSEC 支持。

🔧 How — 怎么做

  1. 登录 Cloudflare 控制台 dash.cloudflare.com
  2. 左侧菜单点击 Domain Registration → Register Domains
  3. 搜索想要的域名
  4. 选择域名和注册年限
  5. 填写注册人信息（联系方式会被 WHOIS 隐私保护隐藏）
  6. 选择支付方式（信用卡或 PayPal），完成付款

注册成功后域名自动出现在 Cloudflare 账户中，DNS 管理即刻可用。

⚠️ 常见问题与注意事项

  • Cloudflare Registrar 支持的 TLD 后缀有限，部分冷门后缀可能不支持
  • 不支持 .cn 等需要实名的国别域名
  • 注册后不支持修改 NS 到非 Cloudflare 服务器（域名锁定在 Cloudflare DNS）
  • 如果需要将域名转出到其他注册商，需要先解锁再获取转移码
  • Cloudflare 域名注册不支持支付宝，需要国际信用卡或 PayPal
