安装 Wordfence + 基础安全扫描

security L2

📖 What — 这是什么

Wordfence Security 是 WordPress 最流行的安全插件（500 万+活跃安装）。它提供防火墙（WAF）、恶意软件扫描、登录安全和实时流量监控四大核心功能。

基础安全扫描会检查 WordPress 核心文件完整性、已知漏洞、恶意代码、过期的插件/主题等安全隐患。

🎯 Why — 为什么要做

WordPress 占全球网站的 40%+，是黑客的首要目标。没有安全插件的 WordPress 站点，就像没锁的门——不是会不会被入侵，而是什么时候被入侵。

安装 Wordfence 是建站后的第一个安全步骤，定期扫描则能及早发现问题。

🔧 How — 怎么做

1. 安装并激活

locwp wp wp-train -- plugin install wordfence --activate

2. 初始配置

  1. 登录后台，进入 Wordfence → Dashboard
  2. 输入邮箱地址接收安全警报
  3. 免费版选择"Free License"，点击继续
  4. 在 Wordfence → Firewall 中，防火墙会自动进入"Learning Mode"，持续一周后自动切换为"Enabled and Protecting"

3. 运行首次扫描

  1. 进入 Wordfence → Scan
  2. 点击 Start New Scan
  3. 等待扫描完成，查看结果报告

扫描会检查：

  • WordPress 核心文件是否被篡改
  • 插件/主题是否有已知漏洞
  • 是否存在可疑文件或后门代码
  • 公开暴露的敏感文件（如 wp-config.php 备份）

4. 处理扫描结果

扫描结果分为 Critical（严重）、Warning（警告）、Info（信息）三级。Critical 必须立即处理，Warning 建议处理。

⚠️ 常见问题与注意事项

  • Local 开发环境中 Wordfence 防火墙可能会误报或影响性能，可以只保留扫描功能
  • 免费版已包含基本防火墙和扫描功能，对于训练环境足够使用
  • 首次扫描可能需要几分钟，取决于站点大小
  • 不要同时安装多个安全插件（如 Wordfence + Sucuri），它们会互相冲突
  • Wordfence 的 WAF 在 Local 环境中可能无法完全工作，因为流量不经过标准 Web 服务器
