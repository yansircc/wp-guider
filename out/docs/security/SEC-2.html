确认注册关闭

security L1

📖 What — 这是什么

WordPress 默认允许任何人注册账号（取决于安装时的设置）。"任何人都可以注册"这个选项在 设置 → 常规 中控制。

对于大多数企业/展示型网站，不需要开放注册。只有管理员手动创建的账号才应该存在。

🎯 Why — 为什么要做

开放注册是最常见的安全隐患之一。垃圾注册机器人会在几小时内创建数百个虚假账号，用于发送垃圾评论、尝试权限提升攻击，或者简单地占用数据库空间。

关闭注册是建站后最基本的安全措施——除非站点确实需要用户注册功能（如社区、会员站）。

🔧 How — 怎么做

方法一：后台操作

  1. 登录后台，进入 设置 → 常规
  2. 找到 成员资格（Membership）选项
  3. 确认 "任何人都可以注册" 未勾选
  4. 保存更改

方法二：WP-CLI

# 查看当前状态（0 = 关闭，1 = 开启）
locwp wp wp-train -- option get users_can_register

# 关闭注册
locwp wp wp-train -- option update users_can_register 0

验证

访问 http://your-site.local/wp-login.php?action=register，应该显示"注册已关闭"或重定向到登录页。

⚠️ 常见问题与注意事项

  • 即使关闭了注册，已有的用户账号不会被删除
  • 部分插件（如 WooCommerce、BuddyPress）有自己的注册系统，需要单独关闭
  • 如果必须开放注册，务必将默认角色设为"订阅者"（Subscriber），并安装验证码插件
  • 定期检查用户列表，删除可疑的注册账号
