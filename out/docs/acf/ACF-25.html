JSON 导入导出与 ACF 工具

ACF L2

本文同时覆盖: ACF-25, ACF-26, ACF-27, ACF-28

📖 What — 这是什么

ACF 支持将字段组配置导出为 JSON 文件，也能从 JSON 文件导入字段组。配合两个社区工具（ACF Playground 和 ACF Helper），可以快速生成和复用字段组配置。

🔧 How — 导出 JSON

  1. 进入 ACF → 工具
  2. 在「导出字段组」区域，勾选要导出的字段组
  3. 点击「导出文件」，下载 JSON 文件

导出的 JSON 包含字段组的完整配置：字段定义、位置规则、显示设置等。不包含字段的实际数据（内容）。

🔧 How — 导入 JSON

  1. 进入 ACF → 工具
  2. 在「导入字段组」区域，选择 JSON 文件
  3. 点击「导入文件」
如果导入的字段组 Key 与已有字段组相同，会覆盖现有配置。导入前确认不会影响已有数据结构。

🛠️ ACF 社区工具

ACF Playground

网址：acf-playground.vercel.app

可视化拖拽工具，在浏览器中通过拖放操作构建字段组，然后导出为 ACF JSON 格式。适合快速搭建字段结构原型。

ACF Helper

网址：acf-helper.vercel.app

AI 辅助工具，用自然语言描述你需要的字段（如"我需要一个产品字段组，包含名称、价格、描述和产品图片"），自动生成 ACF JSON 配置文件。

工作流

  1. 用 ACF Playground 或 ACF Helper 生成 JSON
  2. 在 WordPress 后台通过 ACF 工具页面导入
  3. 根据需要微调字段设置
这两个工具对新手特别有用 — 不需要手动一个个添加字段，直接生成后导入即可。

⚠️ 常见问题与注意事项

  • 版本兼容：JSON 格式与 ACF 版本相关，从旧版导出的 JSON 在新版中导入一般兼容，反之可能有问题
  • 只传配置不传数据：JSON 导入导出只处理字段组的定义（结构），不包含文章中已填写的字段值
  • 团队协作：将 ACF JSON 纳入 Git 版本控制，团队成员可以同步字段组配置
  • 工具生成的 JSON 需要检查：AI 生成的配置可能不完全符合需求，导入后务必在后台验证每个字段的设置
