编辑与查看文章类型设置

ACF L1 NEW

📖 What — 这是什么

创建自定义文章类型（CPT）后，可以随时回到 ACF 后台查看和修改其详细设置，包括标签、可见性、支持的功能、菜单图标等。

🎯 Why — 为什么要做

  • 项目需求变化时需要调整 CPT 的标签或行为
  • 优化后台体验（更换图标、调整菜单位置）
  • 控制前台访问权限（公开/私有）和编辑器功能

🔧 How — 怎么做

  1. 进入 ACF → 文章类型
  2. 点击目标文章类型名称进入编辑页
  3. 常用设置项：
    • 标签：修改复数/单数标签、菜单名称
    • 支持（Supports）：勾选编辑器功能 — 标题、编辑器、缩略图、自定义字段、摘要等
    • 菜单图标：选择 Dashicons 让后台菜单更直观
    • 菜单位置：数字越小越靠上（如 5 在「文章」上方，25 在「评论」下方）
    • 公开可见：控制前台是否可访问
    • 在 REST API 中显示：开启后 Gutenberg 编辑器才能正常工作
  4. 修改后点击保存更改
文章类型 Key（如 product）创建后不可更改。只有标签、图标、功能支持等设置可以随时修改。

⚠️ 常见问题与注意事项

  • REST API 开关：如果用 Gutenberg 编辑器编辑 CPT，必须开启「在 REST API 中显示」，否则编辑器加载不出来
  • Supports 的影响：取消勾选某功能后，编辑界面对应区块会消失，但已保存的数据不会丢失
  • 菜单图标参考：developer.wordpress.org/resource/dashicons/ 查看所有可用图标
