开启 CPT 归档 URL

ACF L2

📖 What — 这是什么

CPT 的归档页（Archive Page）是一个列出该文章类型所有内容的列表页面。例如 yoursite.com/product/ 会展示所有产品。默认情况下，ACF 创建的 CPT 没有开启归档，需要手动启用。

🎯 Why — 为什么要做

  • 不开归档 = 没有产品列表页，用户无法浏览所有产品
  • 归档页是 SEO 的重要入口，搜索引擎可以索引整个内容列表
  • 很多主题和页面构建器依赖归档页来展示 CPT 内容

🔧 How — 怎么做

  1. 进入 ACF → 文章类型，点击要编辑的 CPT
  2. 展开 高级设置（Advanced Settings）
  3. 找到 URL 分区
  4. 将 归档（Has Archive）设为 是
  5. 可选：自定义归档 slug（如 products），默认使用 CPT 的 Key
  6. 点击保存更改
这是 ACF CPT 最常见的坑：很多人创建了 CPT 却发现前台没有列表页，就是因为忘记在 高级 → URL → 归档 这里开启。

保存后刷新固定链接

开启归档后需要刷新 WordPress 的固定链接规则：

  1. 进入 设置 → 固定链接
  2. 不用改任何东西，直接点击保存更改

这一步让 WordPress 重新生成 URL 重写规则，否则归档页可能返回 404。

验证

访问 yoursite.com/product/（或你设置的 slug），应该能看到该 CPT 的文章列表。

⚠️ 常见问题与注意事项

  • 404 错误：开启归档后必须重新保存固定链接，否则会 404
  • 找不到设置：归档选项在「高级设置 → URL」下面，不在顶部的基本设置里
  • 空归档页：如果归档页空白，确认该 CPT 下有已发布的文章
  • 模板覆盖：主题可以通过 archive-{cpt_key}.php 自定义归档页模板
