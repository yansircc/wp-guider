编辑/删除分类法

ACF L2

📖 What — 这是什么

对已有的自定义分类法进行修改或移除。当分类法的标签、绑定关系或层级设置需要调整时，可以编辑；当分类法不再需要时，可以删除整个分类法注册。

🎯 Why — 为什么要做

  • 项目演进过程中可能需要调整分类法的名称或绑定的文章类型
  • 清理不再使用的分类法，保持后台简洁
  • 修改分层设置（从扁平改为分层或反之）

🔧 How — 怎么做

编辑分类法

  1. 进入 ACF → 分类法
  2. 点击要编辑的分类法名称
  3. 可修改的内容包括：
    • 标签（复数/单数名称）
    • 绑定的文章类型
    • 分层设置
    • 高级设置（URL slug、是否公开、是否在 REST API 中可用等）
  4. 修改完成后点击保存更改
分类法 Key 创建后不可更改。如需更改 Key，只能删除并重新创建，已有分类项数据会丢失关联。

删除分类法

  1. 进入 ACF → 分类法
  2. 悬停在要删除的分类法上，点击「移到回收站」

删除分类法注册后，该分类法下的分类项数据仍保留在数据库中（wp_terms 和 wp_term_taxonomy 表），但后台和前台都无法访问。

⚠️ 常见问题与注意事项

  • 修改 Slug 后：记得去 设置 → 固定链接 保存一次，刷新 URL 规则
  • 改分层设置：从分层改为扁平后，已有的父子关系不会自动处理，需手动调整
  • 恢复已删除的分类法：用相同 Key 重建即可恢复关联数据
