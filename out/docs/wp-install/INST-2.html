手动安装 WordPress

WordPress 安装 L2

📖 What — 这是什么

手动安装 WordPress 是指自己完成从下载 WordPress 文件到运行安装向导的全过程。虽然大多数主机都支持一键安装，但理解手动流程对排障和迁移非常有帮助。

手动安装也被称为"著名的五分钟安装"（Famous 5-Minute Install）。

🎯 Why — 为什么要做

有些场景必须手动安装：使用不支持一键安装的主机、VPS 服务器、本地开发环境、或者需要特定版本的 WordPress。理解手动安装流程也能帮助教练更好地理解 WordPress 的底层结构。

🔧 How — 怎么做

前置条件

  • Web 服务器（Apache/Nginx）+ PHP 7.4+ + MySQL 5.7+ 或 MariaDB 10.4+
  • 已创建 MySQL 数据库和用户

安装步骤

  1. 从 wordpress.org/download/ 下载最新版 WordPress（ZIP 格式）
  2. 解压并上传所有文件到服务器的网站根目录（通常是 public_html）
  3. 在浏览器中访问你的域名，WordPress 会自动启动安装向导
  4. 填写数据库信息：
    • 数据库名（Database Name）
    • 用户名（Username）
    • 密码（Password）
    • 数据库主机（通常是 localhost）
    • 表前缀（默认 wp_，建议改为随机前缀如 xk3_）
  5. 点击"提交"，WordPress 会创建 wp-config.php 文件
  6. 填写站点信息：站点标题、管理员用户名、密码、邮箱
  7. 点击"安装 WordPress"，完成

⚠️ 常见问题与注意事项

  • 如果安装向导没有自动启动，检查是否将文件解压到了正确的目录
  • "无法创建 wp-config.php" 错误通常是文件权限问题，确保目录权限为 755
  • "Error establishing a database connection" 说明数据库信息填写错误
  • 表前缀不要使用默认的 wp_，这是安全最佳实践
  • 管理员用户名不要用 "admin"，使用自定义名称增加安全性
