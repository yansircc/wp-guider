自定义链接、子菜单与显示位置

菜单与导航 L2

📖 What — 这是什么

WordPress 菜单系统不仅支持添加站内页面，还支持三个进阶功能：

  • 自定义链接：添加任意 URL 作为菜单项，可指向外部网站
  • 子菜单层级：通过拖拽缩进创建下拉子菜单，形成层级导航
  • 菜单显示位置：将菜单分配到主题提供的位置（如顶部主导航、页脚、侧边栏等）

这三个功能结合使用，可以构建出功能完整的网站导航体系。

🎯 Why — 为什么要做

自定义链接让你可以在菜单中添加外部资源（如社交媒体主页、合作伙伴网站），或创建不指向具体页面的「占位」父菜单项。

子菜单让你在有限的导航栏空间内组织大量页面。例如「服务」下拉显示「网站建设」「SEO 优化」「品牌设计」等子项。

显示位置决定了菜单在前台的呈现方式。同一网站通常需要多个菜单：顶部主导航、页脚链接、移动端菜单等，各自承担不同的导航职责。

🔧 How — 怎么做

添加自定义链接

  1. 在 外观 → 菜单 左侧，展开 自定义链接 面板
  2. 输入 URL（如 https://twitter.com/yourname）
  3. 输入 链接文字（如「Twitter」）
  4. 点击 添加到菜单

创建子菜单层级

  1. 在菜单结构区域，用鼠标拖拽菜单项向右缩进
  2. 缩进的菜单项会变成上方菜单项的子项（前面出现「子项目」标识）
  3. 可以创建多级子菜单，但建议不超过 2 层
  4. 点击 保存菜单

占位父菜单的技巧：如果父菜单项不需要指向任何页面（仅用于展开子菜单），添加一个自定义链接，URL 填 #，文字填你想显示的名称。

设置菜单显示位置

  1. 在菜单编辑页面底部，找到 菜单设置 区域
  2. 在「显示位置」中勾选要分配的位置（如 Primary Menu、Footer Menu）
  3. 可用的位置取决于当前主题，不同主题提供不同的位置选项
  4. 点击 保存菜单

也可以在 外观 → 菜单 → 管理位置 选项卡中，统一查看和管理所有位置的菜单分配。

⚠️ 常见问题与注意事项

  • 外部链接安全：指向外部网站的自定义链接，建议在菜单项详情中勾选「在新标签页中打开」，避免用户离开你的网站
  • 子菜单在移动端的表现：下拉子菜单在桌面端体验良好，但在移动端可能难以操作。确保主题的响应式设计支持触摸展开子菜单
  • # 占位链接的问题：URL 为 # 的菜单项点击后页面会跳到顶部。部分主题会在悬停时显示子菜单，但触屏设备可能需要点击才能展开
  • 切换主题后菜单位置可能丢失：不同主题注册的菜单位置名称不同。切换主题后需要重新分配菜单到新主题的位置
  • 一个位置只能分配一个菜单：但一个菜单可以分配到多个位置
  • 菜单项不显示：最常见的原因是忘记分配显示位置，或分配到了主题未在前台渲染的位置
