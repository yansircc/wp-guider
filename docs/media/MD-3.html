设置特色图片

媒体管理 L2

📖 What — 这是什么

特色图片（Featured Image，也叫缩略图 Thumbnail）是代表文章或页面的封面图片。它不会自动出现在文章正文中，而是由主题决定在哪里、以什么方式展示。

常见的展示位置包括：文章列表页的缩略图、文章顶部的 Banner 大图、社交媒体分享时的预览图（Open Graph Image）、相关文章推荐中的配图等。

🎯 Why — 为什么要做

特色图片是文章的「门面」。在文章列表页中，有特色图片的文章比纯文字文章更吸引点击。当文章被分享到社交媒体时，特色图片会作为预览图展示，直接影响点击率。

没有特色图片的文章在分享时可能显示为空白或使用网站的默认图片，缺乏吸引力。为每篇文章设置相关的特色图片是内容发布的标准流程。

🔧 How — 怎么做

  1. 编辑文章或页面
  2. 在右侧面板中找到 特色图片 区域（通常在面板底部）
  3. 点击 设置特色图片
  4. 从媒体库中选择一张图片，或点击 上传文件 上传新图片
  5. 选中图片后点击 设置特色图片 确认
  6. 更新/发布文章

设置后，右侧面板会显示特色图片的缩略预览。点击图片可以更换或移除。

⚠️ 常见问题与注意事项

  • 特色图片的推荐尺寸：一般建议 1200x630 像素（16:9 比例），兼顾社交媒体分享和网站展示
  • 特色图片 ≠ 文章内图片：设置特色图片后，它不会自动出现在文章正文中（除非主题特别设计了这一点）。如果正文中也需要这张图，还需要手动插入
  • 主题决定展示方式：不同主题对特色图片的显示位置、裁剪方式、比例要求都不同。切换主题后特色图片的显示效果可能改变
  • Alt Text 不要忘记：特色图片同样需要设置 Alt Text，选择图片时可以在媒体库中编辑
