处理网站素材（压缩/裁剪/格式转换）

媒体管理 L2

📖 What — 这是什么

网站素材处理是指在上传图片到 WordPress 之前（或之后），对图片进行压缩、裁剪和格式转换，以确保图片在网站上以最佳状态展示，同时不拖慢页面加载速度。

主要处理手段包括：压缩（减小文件体积）、裁剪/缩放（调整尺寸和比例）、格式转换（如 PNG 转 JPG、JPG 转 WebP）。

🎯 Why — 为什么要做

图片通常是网页中体积最大的资源。未经优化的图片会严重影响页面加载速度——而加载速度直接影响用户体验和搜索排名。Google 的 Core Web Vitals 将加载性能作为排名因素之一。

一张手机拍摄的原始照片可能有 5-10MB，而经过优化后可以压缩到 100-300KB，视觉质量几乎无差别，但文件大小减少了 95% 以上。

🔧 How — 怎么做

上传前处理（推荐）

  1. 调整尺寸：网站用图一般不需要超过 2000 像素宽。用系统自带的图片编辑器或在线工具缩放到合适尺寸
  2. 压缩：使用在线工具如 TinyPNG、Squoosh 压缩图片。有损压缩（Quality 80-85%）可大幅减小体积且肉眼难辨
  3. 格式选择：
    • JPG：照片类图片首选，压缩率高
    • PNG：需要透明背景的图片（Logo、图标）
    • WebP：新一代格式，同等质量下体积比 JPG 小 25-35%，现代浏览器均支持

WordPress 内置裁剪

  1. 在 媒体 → 媒体库 中点击图片
  2. 点击 编辑图片
  3. 可以进行裁剪、旋转、翻转和缩放操作
  4. 选择应用到：所有图片尺寸 / 仅缩略图 / 除缩略图外所有尺寸

使用插件自动优化

安装图片优化插件（如 ShortPixel、Imagify、Smush）可以在上传时自动压缩图片并转换为 WebP 格式。

⚠️ 常见问题与注意事项

  • 不要重复压缩：对已压缩的 JPG 再次压缩会累积质量损失。保留原始文件，只压缩一次
  • WebP 兼容性：WordPress 5.8+ 原生支持 WebP 上传，但 IE11 和部分旧浏览器不支持。优化插件通常会同时保留 JPG 回退
  • 内置裁剪功能有限：WordPress 内置的图片编辑器只能做基础裁剪和缩放，复杂编辑（调色、抠图）需使用专业工具
  • 批量优化：如果网站已有大量未优化的图片，使用优化插件的「批量优化」功能可一次性处理所有历史图片
