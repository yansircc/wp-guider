提交 Sitemap 到 GSC

Google Analytics L2

📖 What — 这是什么

XML 站点地图（Sitemap）是一个列出网站所有公开 URL 的文件，帮助搜索引擎发现和抓取你的页面。Yoast SEO 和 SEOPress 都会自动生成站点地图，你只需将其地址提交给 Google Search Console。

提交站点地图不保证 Google 一定收录所有页面，但可以加速发现新页面和更新。

🎯 Why — 为什么要做

Google 的爬虫（Googlebot）虽然可以自行发现页面，但对新站点或内部链接不完善的站点，很多页面可能长时间不被收录。站点地图是主动告诉 Google "这些页面存在，请来抓取"的最高效方式。

🔧 How — 怎么做

1. 确认站点地图 URL

  • Yoast SEO：https://your-site.com/sitemap_index.xml
  • SEOPress：https://your-site.com/sitemaps.xml

在浏览器中访问上述地址，确认能看到 XML 格式的站点地图。

2. 在 GSC 中提交

  1. 登录 Google Search Console
  2. 在左侧菜单点击 站点地图（Sitemaps）
  3. 在「添加新的站点地图」输入框中粘贴站点地图 URL
  4. 点击 提交
  5. 状态显示为「成功」即表示 Google 已接收
只需提交一次。SEO 插件会在内容更新时自动更新站点地图，Google 会定期重新抓取。

⚠️ 常见问题与注意事项

  • 状态「无法获取」：检查站点地图 URL 是否正确、网站是否可被外部访问、是否有安全插件阻止了爬虫
  • 站点地图中没有某个页面：检查该页面是否被设为 noindex，或者是草稿/私密状态。只有已发布且未设 noindex 的内容才会出现在站点地图中
  • 不要提交多个重复站点地图：如果同时启用了 Yoast 和 SEOPress，会生成两份不同的站点地图，只提交你正在使用的 SEO 插件生成的那个
  • 提交后不代表立即收录：Google 会按自己的节奏抓取，通常几天到几周不等
  • 固定链接影响站点地图：如果固定链接设为「朴素」模式，站点地图 URL 可能不正确。确保使用「文章名」等非朴素模式
