用 DoLogin 生成免密登录链接

用户管理 L2

📖 What — 这是什么

DoLogin Security 是一款 WordPress 插件，核心功能之一是生成临时的免密登录链接（Magic Login Link）。点击链接即可自动登录指定的 WordPress 用户账号，无需输入密码。

链接可以设置有效期和使用次数限制，过期后自动失效。

🎯 Why — 为什么要做

在建站外包和培训场景中，经常需要让客户或学员快速登录后台，而不想每次都传递明文密码。免密链接更安全、更方便：

  • 给客户一个一次性链接查看网站后台，不必泄露密码
  • 培训时让学员快速进入指定账号开始操作
  • 远程协助时分享给技术支持人员

🔧 How — 怎么做

  1. 安装并激活 DoLogin Security 插件（插件 → 安装插件 → 搜索 "DoLogin"）
  2. 进入 用户 → 所有用户
  3. 在目标用户的操作列中，找到 DoLogin Link 或类似按钮
  4. 点击后会生成一个包含 token 的 URL
  5. 复制该链接发送给需要登录的人

也可以通过插件设置页面批量生成和管理登录链接。

⚠️ 常见问题与注意事项

  • 链接默认有有效期限制，过期后点击无效，需要重新生成
  • 不要在公开渠道（如社交媒体）分享免密链接，应通过私信或邮件传递
  • 管理员账号的免密链接更要谨慎使用，建议限制为一次性使用
  • 如果链接失效或无法登录，检查插件设置中的过期时间和使用次数限制
  • 部分安全插件可能拦截免密登录行为，需要在白名单中排除
