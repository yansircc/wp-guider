修改昵称和显示名

用户管理 L1

📖 What — 这是什么

WordPress 用户有多个名称字段：用户名（登录名，不可修改）、昵称（Nickname）、公开显示名称（Display name）。前台显示的作者名是"公开显示名称"，而非用户名。

默认情况下，公开显示名称与用户名相同，出于安全考虑应立即修改，避免暴露登录用户名。

🎯 Why — 为什么要做

WordPress 的登录用户名一旦暴露，攻击者只需要猜密码即可尝试暴力登录。如果前台作者页显示的就是登录用户名，等于把一半的登录凭证公开了。

修改显示名称为昵称或真实姓名，既能保护安全，又能让前台显示更专业。

🔧 How — 怎么做

  1. 登录 WordPress 后台
  2. 进入 用户 → 我的个人资料（Users → Profile）
  3. 在 昵称（Nickname）字段中输入一个不同于用户名的名字
  4. 在 公开显示为（Display name publicly as）下拉菜单中选择刚设置的昵称
  5. 滚动到底部，点击 更新个人资料

如果要修改其他用户的显示名，进入 用户 → 所有用户，点击该用户进行编辑。

⚠️ 常见问题与注意事项

  • 用户名（Username）创建后无法修改，只能通过创建新用户再删旧用户来变更
  • 公开显示名称会出现在文章作者栏、评论区、作者归档页等位置
  • 建议所有管理员账号在建站后第一时间修改显示名称
