修改默认角色

用户管理 L1

📖 What — 这是什么

WordPress 的"新用户默认角色"决定了通过注册页面注册的新用户会被分配什么角色。默认值是"订阅者"（Subscriber），这是权限最低的角色。

此设置位于 设置 → 常规 页面，与用户注册开关在同一区域。

🎯 Why — 为什么要做

如果站点需要开放注册（如会员站、论坛），需要确认默认角色设置正确。将默认角色设为"订阅者"是最安全的选择，因为订阅者只能管理自己的个人资料。

如果不小心将默认角色设为"管理员"或"编辑"，任何注册用户都会获得高级权限，后果非常严重。

🔧 How — 怎么做

  1. 登录 WordPress 后台
  2. 进入 设置 → 常规（Settings → General）
  3. 找到 新用户默认角色（New User Default Role）下拉菜单
  4. 确认设置为 订阅者（Subscriber）
  5. 点击 保存更改

WordPress 内置角色权限一览

  • 管理员（Administrator）— 完全控制权
  • 编辑（Editor）— 管理所有文章和页面
  • 作者（Author）— 发布和管理自己的文章
  • 投稿者（Contributor）— 写文章但不能发布
  • 订阅者（Subscriber）— 只能管理个人资料

⚠️ 常见问题与注意事项

  • 即使关闭了注册功能，也建议将默认角色保持为"订阅者"，作为安全兜底
  • 部分安全漏洞可能绕过注册开关创建用户，低权限的默认角色可以减轻风险
  • 此设置不影响管理员手动创建用户时选择的角色
