创建三个页面并发布

页面管理 L1

📖 What — 这是什么

WordPress 中的「页面」（Page）是一种静态内容类型，与文章（Post）不同，页面不按时间排序，通常用于承载固定内容，如「关于我们」「服务介绍」「联系方式」等。

创建并发布页面是搭建网站的基础操作。一个完整的企业网站通常至少需要首页、关于页、联系页三个核心页面。页面发布后，访客即可通过 URL 直接访问。

页面使用区块编辑器（Block Editor / Gutenberg）编辑内容，支持段落、标题、图片、列表等多种区块类型。

🎯 Why — 为什么要做

页面是网站的骨架。没有页面，网站就只是一个空壳。企业客户、访客需要通过页面了解你的业务、服务和联系方式。

批量创建核心页面是建站的第一步，也是后续设置导航菜单、配置首页的前提条件。尽早完成页面创建，才能推进整体网站搭建流程。

🔧 How — 怎么做

  1. 进入 WordPress 后台，点击左侧菜单 页面 → 添加页面
  2. 在标题栏输入页面标题，例如「关于我们」
  3. 在编辑区域添加内容：点击 + 按钮插入段落、标题等区块
  4. 编辑完成后，点击右上角 发布 按钮
  5. 在弹出的确认面板中再次点击 发布
  6. 重复以上步骤，分别创建「服务介绍」和「联系我们」页面

创建完成后，可在 页面 → 所有页面 中查看已发布的页面列表。

⚠️ 常见问题与注意事项

  • 页面标题不要重复：虽然 WordPress 允许同名页面，但会导致 slug 冲突（自动追加 -2、-3），影响 SEO 和用户体验
  • 发布前预览：点击「预览」按钮可在新窗口中查看页面效果，确认内容和排版无误再发布
  • 注意发布状态：如果只点了「保存草稿」而未点「发布」，页面不会对外可见
  • 固定链接：页面发布后 WordPress 会根据标题自动生成 slug，中文标题建议手动修改为英文 slug
