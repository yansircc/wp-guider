设置时区

站点设置 L1

📖 What — 这是什么

WordPress 的时区设置决定了站点中所有时间相关功能使用的基准时区，包括文章发布时间、定时发布、评论时间戳、wp-cron 计划任务等。

该设置位于 设置 → 常规 页面。可以选择具体城市（如 Shanghai）或 UTC 偏移量。

🎯 Why — 为什么要做

如果时区设置不正确，定时发布的文章会在错误的时间上线，后台显示的时间也会与实际不符。对于面向中国客户的站点，应设置为 Asia/Shanghai（UTC+8）。

外贸站通常根据目标市场设置时区，比如面向美国客户设置为美东时间。

🔧 How — 怎么做

  1. 登录 WordPress 后台
  2. 进入 设置 → 常规（Settings → General）
  3. 找到 时区（Timezone）下拉菜单
  4. 选择对应的城市，如 Shanghai（位于 Asia 分类下）
  5. 点击 保存更改

保存后页面下方会显示当前本地时间和 UTC 时间，用于验证设置是否正确。

⚠️ 常见问题与注意事项

  • 建议选择城市名称而非 UTC 偏移量，因为城市名称会自动处理夏令时
  • 时区设置错误会导致 wp-cron 任务（如自动备份、定时发布）在错误时间执行
  • 修改时区不会影响已发布文章的时间戳显示
