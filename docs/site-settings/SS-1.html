修改站点标题

站点设置 L1

📖 What — 这是什么

站点标题（Site Title）是 WordPress 站点的名称，显示在浏览器标签页、搜索引擎结果、以及主题的页眉区域。它是访客对网站的第一印象，也是 SEO 的重要信号之一。

WordPress 安装时会要求填写站点标题，但安装后随时可以在后台修改。标题会影响所有页面的 <title> 标签输出。

🎯 Why — 为什么要做

客户的品牌名称、业务定位可能会调整，站点标题需要随之更新。一个准确的站点标题能帮助搜索引擎理解网站主题，也能让访客快速识别网站身份。

如果使用默认的安装标题（如 "Just another WordPress site"），会显得不专业，影响品牌形象和 SEO 表现。

🔧 How — 怎么做

  1. 登录 WordPress 后台
  2. 进入 设置 → 常规（Settings → General）
  3. 在 站点标题（Site Title）字段中输入新的标题
  4. 滚动到页面底部，点击 保存更改

保存后，刷新前台页面即可看到浏览器标签页和页眉中的标题已更新。

⚠️ 常见问题与注意事项

  • 如果前台标题没有变化，可能是缓存导致，清除浏览器缓存或站点缓存后再查看
  • 部分主题在 Customizer 中也有标题设置，两处需保持一致
  • 站点标题不宜过长，建议控制在 20 个字符以内
  • 安装了 SEO 插件（如 Yoast）时，SEO 标题模板可能会覆盖默认标题显示格式
