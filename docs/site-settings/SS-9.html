关闭评论

站点设置 L1

📖 What — 这是什么

WordPress 默认开启文章评论功能，允许访客在文章下方留言。对于企业站、产品展示站等不需要评论互动的网站，应该全局关闭评论功能。

关闭评论涉及两个层面：全局默认设置（影响新文章）和已有文章的评论状态。

🎯 Why — 为什么要做

开放评论是垃圾评论（spam）的主要入口。即使安装了 Akismet 等反垃圾插件，每天仍会收到大量垃圾评论，浪费数据库资源。对于不需要评论的网站，直接关闭最省心。

垃圾评论中常包含恶意链接，影响站点 SEO 和安全性。

🔧 How — 怎么做

  1. 登录 WordPress 后台
  2. 进入 设置 → 讨论（Settings → Discussion）
  3. 取消勾选 "允许他人在新文章上发表评论"（Allow people to submit comments on new posts）
  4. 点击 保存更改
注意：这只影响新发布的文章。已有文章的评论仍然开启，需要单独处理。

关闭已有文章的评论

  1. 进入 文章 → 所有文章
  2. 勾选顶部全选框选中所有文章
  3. 在 批量操作 下拉菜单中选择 编辑，点击 应用
  4. 在批量编辑面板中，将 评论 改为 不允许
  5. 点击 更新

⚠️ 常见问题与注意事项

  • 页面（Page）的评论默认也是开启的，需要在页面列表中同样批量关闭
  • 关闭评论后，已有的历史评论不会被删除，仍然会显示在前台
  • 如果想彻底隐藏评论区域，可能还需要在主题模板中处理
  • 使用 Disable Comments 插件可以一键全局禁用，比手动操作更彻底
