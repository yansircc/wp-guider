关闭注册

站点设置 L1

📖 What — 这是什么

WordPress 默认允许用户通过 /wp-login.php?action=register 页面自行注册账号。对于大多数企业站和展示站，这个功能是不需要的，应该关闭。

"任何人都可以注册"（Anyone can register）是 WordPress 常规设置中的一个复选框，控制是否开放前台注册。

🎯 Why — 为什么要做

开放注册会带来安全风险：垃圾注册机器人会大量注册账号，占用数据库资源，甚至可能被利用进行攻击。对于不需要会员功能的网站，关闭注册是基本的安全措施。

即使需要用户注册（如电商、会员站），也应通过专用插件（如 WooCommerce）来管理，而非使用 WordPress 默认注册功能。

🔧 How — 怎么做

  1. 登录 WordPress 后台
  2. 进入 设置 → 常规（Settings → General）
  3. 找到 成员资格（Membership）区域
  4. 确保 "任何人都可以注册"（Anyone can register）复选框未勾选
  5. 点击 保存更改

⚠️ 常见问题与注意事项

  • 关闭注册后，管理员仍可在后台 用户 → 添加用户 手动创建账号
  • 如果安装了 WooCommerce，客户注册由 WooCommerce 设置控制，与此设置独立
  • 建议配合安全插件（如 Wordfence）限制登录页面访问，进一步降低风险
