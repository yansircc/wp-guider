切换语言

站点设置 L1

📖 What — 这是什么

WordPress 支持多语言界面，可以在后台设置中切换站点语言。切换语言会影响后台管理界面的显示语言，以及部分前台元素（如日期格式、搜索提示文字等）。

WordPress 官方翻译团队维护了数十种语言包，切换时会自动下载对应的翻译文件。

🎯 Why — 为什么要做

教练经常需要为不同客户配置站点：中文客户需要简体中文后台，外贸客户可能需要英文后台。掌握语言切换是基本操作。

训练环境建议使用英文界面，因为绝大多数 WordPress 教程、论坛、插件文档都以英文为主，使用英文后台能更快找到对应设置。

🔧 How — 怎么做

  1. 登录 WordPress 后台
  2. 进入 设置 → 常规（Settings → General）
  3. 找到 站点语言（Site Language）下拉菜单
  4. 选择目标语言（如 English (United States) 或 简体中文）
  5. 点击 保存更改，WordPress 会自动下载语言包

注意：站点语言和用户个人语言是独立的。每个用户可以在 用户 → 我的个人资料 中设置自己的后台语言，不影响其他用户。

⚠️ 常见问题与注意事项

  • 切换语言不会翻译已有的文章内容，只影响界面文字
  • 部分插件可能没有对应语言的翻译，切换后这些插件菜单仍显示英文
  • 如果需要建多语言网站（前台多语言），需要使用 WPML 或 Polylang 等插件，不能靠切换站点语言实现
