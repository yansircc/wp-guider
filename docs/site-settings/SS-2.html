修改副标题

站点设置 L1

📖 What — 这是什么

副标题（Tagline）是对站点的简短描述，通常显示在站点标题下方。WordPress 默认的副标题是 "Just another WordPress site"。

副标题会出现在浏览器标签页（与标题组合）、RSS 订阅源以及部分主题的页眉区域。搜索引擎也会参考副标题来理解网站内容。

🎯 Why — 为什么要做

默认副标题非常不专业，必须修改。副标题是向访客和搜索引擎传达网站定位的好机会。比如一个外贸站可以写"Professional LED Lighting Manufacturer"，一目了然。

如果不需要副标题，也可以留空。但绝对不能保留默认值。

🔧 How — 怎么做

  1. 登录 WordPress 后台
  2. 进入 设置 → 常规（Settings → General）
  3. 在 副标题（Tagline）字段中输入新的描述，或清空
  4. 点击 保存更改

如果不想在前台显示副标题，可以留空该字段。但注意有些主题会在留空时仍显示占位文本，这种情况需要在 Customizer 中处理。

⚠️ 常见问题与注意事项

  • 务必在建站初期就修改掉默认副标题，避免被搜索引擎收录默认文本
  • 副标题建议简短有力，10-15 个英文单词或 15-20 个中文字
  • SEO 插件可能会在搜索结果中使用副标题，确保内容准确且包含关键词
