添加与编辑自定义文章类型

ACF L1

本文同时覆盖: ACF-2, ACF-3

📖 What — 这是什么

自定义文章类型（Custom Post Type, CPT）让你在 WordPress 默认的「文章」和「页面」之外，创建全新的内容类型。例如：产品、案例、团队成员、常见问题等。ACF Pro 提供了可视化界面来注册和管理 CPT，无需写一行代码。

🎯 Why — 为什么要做

  • 默认的文章/页面无法满足结构化内容需求（如产品需要价格、规格等字段）
  • CPT 让不同类型的内容拥有独立的管理入口和模板
  • 搭配自定义字段和分类法，可以构建完整的内容管理系统

🔧 How — 怎么做

创建 CPT

  1. 进入 ACF → 文章类型
  2. 点击「添加新文章类型」
  3. 填写关键信息：
    • 复数标签：如「产品」（用于菜单显示）
    • 单数标签：如「产品」
    • 文章类型 Key：如 product（仅英文小写 + 下划线，创建后不可更改）
  4. 点击保存更改
Key 是程序内部标识，一旦创建不可修改。建议用简短的英文单数形式，如 product、case_study、faq。

编辑 CPT 设置

创建后可随时回到 ACF → 文章类型，点击已有的 CPT 进入编辑。常用设置：

  • 支持（Supports）：勾选需要的编辑器功能 — 标题、编辑器、缩略图、自定义字段等
  • 菜单图标：选择 Dashicons 图标，让后台菜单更直观
  • 公开可见：设置前台是否可以访问该 CPT 的页面
  • 是否有归档：是否生成列表页（详见归档 URL 文档）

验证

创建成功后，WordPress 后台左侧菜单会出现新的 CPT 入口（如「产品」），点击可以开始添加内容。

⚠️ 常见问题与注意事项

  • Key 不可更改：文章类型 Key 一旦保存就无法修改，已有数据也绑定到这个 Key，请提前规划好命名
  • 与 CPT UI 区别：ACF Pro 内置了 CPT 注册功能，不需要额外安装 CPT UI 插件。两者功能类似，选一个用即可
  • slug 冲突：CPT 的 Key 不要和 WordPress 内建的保留名冲突（如 post、page、attachment、category、tag）
  • 数量建议：一般项目 3-5 个 CPT 足够，避免过度创建
