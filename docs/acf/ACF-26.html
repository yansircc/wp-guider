导入 ACF JSON

ACF L2

📖 What — 这是什么

ACF 支持从 JSON 文件导入字段组配置，将字段组的完整结构（字段定义、位置规则、显示设置）一次性导入到 WordPress 站点中。这是团队协作和站点迁移的核心功能。

🎯 Why — 为什么要做

  • 在新站点快速复制已有的字段组结构
  • 团队成员间同步 ACF 配置
  • 导入由 ACF Playground 或 ACF Helper 工具生成的配置

🔧 How — 怎么做

  1. 进入 ACF → 工具
  2. 找到「导入字段组」区域
  3. 点击选择文件，选择 .json 文件
  4. 点击「导入文件」
  5. 成功后页面会提示导入了哪些字段组

验证

导入后进入 ACF → 字段组，确认新字段组出现在列表中。点击进入检查字段定义和位置规则是否正确。

如果导入的字段组 Key 与已有字段组相同，会覆盖现有配置。导入前检查 JSON 文件中的 "key" 字段，确认不会意外覆盖。

⚠️ 常见问题与注意事项

  • 只导入结构不导入数据：JSON 包含字段组定义，不包含文章中已填写的字段值
  • 版本兼容：从高版本 ACF 导出的 JSON 在低版本中导入可能不兼容
  • 位置规则检查：导入后检查位置规则引用的文章类型或页面是否在当前站点存在
  • 批量导入：一个 JSON 文件可以包含多个字段组，一次导入全部
