ACF Helper — 自然语言生成 ACF JSON

ACF L2 NEW

📖 What — 这是什么

ACF Helper（acf-helper.vercel.app）是一个 AI 辅助工具，用自然语言描述你需要的字段组，自动生成可直接导入 ACF 的 JSON 配置文件。无需手动逐个配置字段。

🎯 Why — 为什么要做

  • 大幅加速字段组搭建速度（从手动十几分钟缩短到几十秒）
  • 降低新手门槛 — 用中文描述需求即可
  • 生成的 JSON 格式标准，直接导入无需调整

🔧 How — 怎么做

  1. 访问 acf-helper.vercel.app
  2. 在输入框中用自然语言描述需求，例如： 
    创建一个「团队成员」字段组，包含：
- 姓名（文本，必填）
- 职位（文本）
- 照片（图片）
- 简介（文本区域）
- 邮箱（URL）
- 是否显示在首页（True/False）
  3. 点击生成，获取 JSON 配置
  4. 复制 JSON 内容保存为 .json 文件
  5. 在 WordPress 后台 ACF → 工具 → 导入 导入该文件
描述越具体，生成结果越准确。建议写明字段类型、是否必填、选项值等细节。

⚠️ 常见问题与注意事项

  • 导入后务必检查：AI 生成的配置可能有细节偏差，导入后逐一检查字段类型、名称和设置
  • 位置规则需手动设置：工具可能不会生成位置规则，导入后需要手动指定字段组出现在哪个文章类型
  • Field Name 检查：确认生成的字段名符合命名规范（英文小写 + 下划线），且不与已有字段冲突
  • 搭配 ACF Playground：如果更喜欢可视化操作，可以用 acf-playground.vercel.app 拖拽生成
