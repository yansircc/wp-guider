日期选择器 / 颜色选择器字段

ACF L3

📖 What — 这是什么

  • 日期选择器（Date Picker）：弹出日历控件选择日期
  • 日期时间选择器（Date Time Picker）：日期 + 时间
  • 颜色选择器（Color Picker）：可视化选色板选择颜色值

这些字段提供专用的输入控件，确保数据格式统一且操作友好。

🎯 Why — 为什么要做

  • 日期字段避免格式混乱（「2024/1/1」vs「1月1日」vs「Jan 1」）
  • 颜色字段确保输出标准的 hex 值，可直接用于 CSS
  • 可视化控件比手动输入更高效、更不容易出错

🔧 How — 怎么做

日期选择器

  1. 添加字段，选择类型「日期选择器」
  2. 关键配置：
    • 显示格式：编辑界面看到的格式（如 Y年m月d日）
    • 返回格式：程序获取的格式（如 Y-m-d → 2024-03-15）
    • 一周开始于：日历第一列是周一还是周日

颜色选择器

  1. 添加字段，选择类型「颜色选择器」
  2. 配置：
    • 默认值：预设颜色（如 #333333）
    • 支持 RGBA：开启后可以选择带透明度的颜色

实际例子

  • 活动开始日期：日期选择器，返回格式 Y-m-d
  • 产品标签颜色：颜色选择器，默认 #3b82f6，用于前端自定义标签样式

⚠️ 常见问题与注意事项

  • 显示格式 ≠ 返回格式：编辑者看到的和前端获取的可以不同，返回格式建议用标准格式 Y-m-d
  • 时区问题：日期选择器不包含时区信息，结合 WordPress 时区设置使用
  • 颜色值：不开 RGBA 时返回 hex（如 #ff0000），开了返回 rgba（如 rgba(255,0,0,0.5)）
  • 日期排序：如果需要按日期排序查询，返回格式必须用 Ymd（无分隔符），这是 WordPress meta_query 排序的要求
