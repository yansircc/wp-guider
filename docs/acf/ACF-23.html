分类法字段

ACF L3

📖 What — 这是什么

ACF 的分类法字段（Taxonomy Field）让你在文章编辑界面中以自定义的方式选择分类项。与 WordPress 默认的分类选择框不同，ACF 分类法字段提供更灵活的显示形式和更多控制选项。

🎯 Why — 为什么要做

  • 默认的分类法面板外观固定，有时不符合编辑需求
  • ACF 分类法字段可以自定义外观（下拉框、复选框、多选等）
  • 可以精确控制选择行为：是否允许新建分类项、是否保存为文章的实际分类关系

🔧 How — 怎么做

  1. 编辑字段组，点击「+ 添加字段」
  2. 字段类型选择「分类法」
  3. 配置关键选项：
    • 分类法：选择要关联的分类法（如「产品分类」、「标签」）
    • 外观：复选框、多选框、单选按钮或下拉菜单
    • 允许添加新分类：编辑者是否可以在此直接新建分类项
    • 保存分类：是否将选择结果保存为 WordPress 原生的文章-分类关系
  4. 保存字段组
「保存分类」选项很关键：开启后选择的分类会同步到 WordPress 的原生分类关系中，关闭则只存在 ACF 的自定义字段中。建议开启，否则前端查询和归档页可能找不到这些分类关系。

⚠️ 常见问题与注意事项

  • 重复显示：如果用了 ACF 分类法字段，建议隐藏 WordPress 默认的分类法面板（在字段组设置中勾选「隐藏界面元素 → 分类法」），避免两个分类选择框同时出现
  • 返回值格式：可选「分类项对象」或「分类项 ID」，前端开发时注意匹配
  • 与原生分类的区别：ACF 分类法字段是在字段组中添加的，位置和外观可控；原生分类面板固定在右侧栏
