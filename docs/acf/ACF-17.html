选择框 / 复选框 / 单选按钮字段

ACF L2

📖 What — 这是什么

ACF 的选择类字段用于从预定义选项中选择值：

  • 选择框（Select）：下拉菜单，可选单选或多选
  • 复选框（Checkbox）：多个选项同时勾选
  • 单选按钮（Radio Button）：只能选一个选项

🎯 Why — 为什么要做

  • 限定输入范围，确保数据一致性（避免编辑者自由输入造成格式混乱）
  • 选项固定时比文本输入更高效、更不容易出错
  • 便于前端按选项值进行筛选和分类展示

🔧 How — 怎么做

选项格式

三种字段的选项设置方式相同，每行一个选项：

red : 红色
blue : 蓝色
green : 绿色

冒号左边是存储值（英文），右边是显示标签（中文）。如果不用冒号，存储值和标签相同。

选择框（Select）

  • 适合选项较多（5+）的场景
  • 可开启多选（Allow Multiple）
  • 可开启 AJAX 搜索（选项很多时提升性能）
  • 可设空值提示（如「请选择...」）

复选框（Checkbox）

  • 适合需要多选且选项较少（2-8）的场景
  • 所有选项直接展示，勾选即可
  • 可设布局：垂直或水平排列
  • 可设「全选」切换按钮

单选按钮（Radio）

  • 适合必须且只能选一个的场景
  • 可设布局：垂直或水平
  • 可设「其他」选项让用户自由输入

实际例子

产品状态字段（单选按钮）：

in_stock : 有货
pre_order : 预售
out_of_stock : 缺货

⚠️ 常见问题与注意事项

  • 选哪种：单选用 Radio 或 Select，多选用 Checkbox 或 Select（多选模式）
  • 存储值用英文：方便程序处理和国际化，显示标签用中文
  • 修改选项：修改存储值后，已有数据不会自动更新，旧值变成无效选项
  • 默认值：可以设置默认选中的选项
