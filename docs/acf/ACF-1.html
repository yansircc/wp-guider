安装 ACF Pro

ACF L1

📖 What — 这是什么

Advanced Custom Fields Pro（ACF Pro）是 WordPress 最常用的自定义字段插件。Pro 版额外提供自定义文章类型（CPT）、分类法（Taxonomy）注册、重复器字段、灵活内容等高级功能。从 2023 年起 ACF 归属 WP Engine，Pro 版需要订阅授权。

注意：我们使用的是 ACF Pro，不是免费版。免费版不包含 CPT/Taxonomy 注册功能。

🎯 Why — 为什么要做

  • 后续的自定义文章类型、分类法、字段组操作都依赖 ACF Pro
  • ACF Pro 提供可视化界面来创建 CPT 和 Taxonomy，无需写代码
  • 几乎所有中大型 WordPress 项目都会用到自定义字段，ACF 是行业标准

🔧 How — 怎么做

方法一：上传 zip 安装

  1. 从 ACF 官网（advancedcustomfields.com）下载 Pro 版 zip 文件
  2. 进入 WordPress 后台 → 插件 → 安装插件 → 上传插件
  3. 选择下载的 zip 文件，点击立即安装
  4. 安装完成后点击启用插件

方法二：通过授权码激活

  1. 安装完成后进入 ACF → 更新
  2. 输入 License Key 激活，确保可以接收后续更新

验证安装成功

安装成功后，WordPress 后台左侧菜单会出现 ACF 主菜单，展开后可以看到：

  • 字段组（Field Groups）
  • 文章类型（Post Types）— 仅 Pro 版
  • 分类法（Taxonomies）— 仅 Pro 版

⚠️ 常见问题与注意事项

  • 免费版 vs Pro 版：如果菜单里没有「文章类型」和「分类法」选项，说明装的是免费版，需要升级
  • PHP 版本：ACF Pro 最新版要求 PHP 7.4+，确保主机环境满足
  • 与 CPT UI 的区别：ACF Pro 自带 CPT 注册功能，不需要另外安装 CPT UI 插件
  • 激活问题：每个 License Key 有站点数量限制，超出需要在 ACF 官网管理已激活站点
