ACF 辅助工具

ACF L2

本文同时覆盖: ACF-27, ACF-28

📖 What — 这是什么

两个在线工具，帮助你快速生成 ACF 字段组的 JSON 配置文件，省去在 WordPress 后台手动创建字段的时间。

  • ACF Playground：可视化拖拽界面，像搭积木一样构建字段组
  • ACF Helper：用自然语言描述需求，AI 自动生成 ACF JSON

🎯 Why — 为什么要做

  • 大批量创建字段时，在后台逐个添加效率低
  • 用工具生成 JSON 后直接导入，比手动操作快 5-10 倍
  • 减少配置错误，工具会自动处理字段 Key 和格式

🔧 How — 怎么做

ACF Playground（可视化拖拽）

访问 acf-playground.vercel.app

  1. 在左侧面板拖拽字段类型到中间区域
  2. 点击字段进行配置（标签、名称、选项等）
  3. 调整字段顺序
  4. 点击导出 JSON 下载配置文件
  5. 回到 WordPress ACF → 工具 → 导入，上传 JSON 文件

ACF Helper（自然语言生成）

访问 acf-helper.vercel.app

  1. 用中文或英文描述你的需求，例如： 
    我需要一个「产品信息」字段组，包含：
- 产品价格（数字，必填，最小0）
- 产品主图（图片，必填）
- 产品状态（单选：在售/预售/下架）
- 购买链接（URL）
应用于「product」文章类型
  2. AI 会生成完整的 ACF JSON 配置
  3. 检查生成结果，按需微调
  4. 复制 JSON 或下载文件
  5. 导入到 WordPress

⚠️ 常见问题与注意事项

  • 导入前检查：工具生成的 JSON 建议先检查字段 Key 命名是否符合你的规范
  • 版本兼容：确认生成的 JSON 格式与你的 ACF Pro 版本兼容
  • 覆盖风险：如果 Key 与站点已有字段组相同，导入会覆盖
  • 两个工具搭配：复杂需求用 Helper 生成初始版本，再用 Playground 微调
