添加第二个网站 (多站点管理)

主机 L2

📖 What — 这是什么

Hostinger 的 Business 及以上方案支持在同一个主机账号下托管多个独立网站。每个网站有自己的域名、WordPress 安装和独立数据。这不是 WordPress Multisite，而是主机层面的多站点管理。

🎯 Why — 为什么要做

建站服务商通常需要在一个 Hostinger 账号下为多个客户建站。掌握添加新网站的流程，可以高效管理多个项目，避免为每个站单独购买主机。

也适用于同一客户有多个品牌/业务需要独立网站的场景。

🔧 How — 怎么做

  1. 登录 Hostinger hPanel
  2. 点击 Websites（或首页的 Add Website 按钮）
  3. 点击 Create or migrate a website
  4. 选择 WordPress 作为建站平台
  5. 输入新网站的 WordPress 管理员邮箱和密码
  6. 绑定域名：
    • 使用已有域名：输入域名并配置 DNS
    • 临时域名：Hostinger 提供免费临时域名用于测试
  7. 选择数据中心（建议与第一个网站相同）
  8. 等待安装完成（通常 1-2 分钟）

⚠️ 常见问题与注意事项

  • 共享主机上的多个网站共享同一台服务器资源，某个站流量暴涨可能影响其他站
  • Premium 方案最多 100 个站点，Business 方案同样是 100 个，但资源更充裕
  • 每个网站有独立的 WordPress 后台和数据库，互不干扰
  • 如果网站数量多或流量大，建议升级到 Cloud 方案获得更好的性能隔离
  • 临时域名仅用于开发测试，上线前需要绑定正式域名
