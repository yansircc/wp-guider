UpdraftPlus 备份

backup-maintenance L2

📖 What — 这是什么

UpdraftPlus 是 WordPress 最流行的备份插件，拥有超过 300 万活跃安装量。它能一键备份整个站点的文件和数据库，并支持将备份发送到 Google Drive、Dropbox、Amazon S3 等远程存储。

WordPress 本身没有内置完整的备份功能，UpdraftPlus 填补了这个关键空缺。备份内容包含五个部分：数据库、插件、主题、上传文件和其他文件，可以按需选择。

🎯 Why — 为什么要做

没有备份等于裸奔。服务器故障、插件冲突白屏、黑客入侵、误删内容——这些都不是"会不会发生"的问题，而是"什么时候发生"的问题。

有备份，灾难只是小插曲；没备份，任何意外都可能是毁灭性的。备份是网站管理的第一铁律，而 UpdraftPlus 让这件事变得简单。

🔧 How — 怎么做

1. 安装并激活

locwp wp wp-train -- plugin install updraftplus --activate

或通过后台：插件 → 安装插件，搜索 "UpdraftPlus"，点击安装并激活。

2. 执行手动备份

  1. 进入 设置 → UpdraftPlus Backups
  2. 点击 立即备份（Backup Now）
  3. 勾选需要备份的内容（建议首次全选）
  4. 点击确认，等待备份完成

3. 备份文件说明

UpdraftPlus 会在 wp-content/updraft/ 目录下生成多个压缩文件：

  • backup_*-db.gz — 数据库备份
  • backup_*-plugins.zip — 插件文件
  • backup_*-themes.zip — 主题文件
  • backup_*-uploads.zip — 上传的媒体文件
  • backup_*-others.zip — 其他 wp-content 文件
备份完成后，在"现有备份"列表中点击验证，确认备份文件完整无损。

⚠️ 常见问题与注意事项

  • 备份只存本地等于没有备份——服务器挂了备份一起丢，建议配置远程存储
  • 免费版支持本地存储和基本云存储（Google Drive、Dropbox 等），足够日常使用
  • 大站点的 uploads 目录可能非常大，备份和恢复耗时较长
  • 训练环境中更推荐 WP Reset 的快照功能——更轻量、恢复更快
  • 首次安装后建议立即做一次完整备份，验证流程是否通畅
