配置 SSL 证书

WordPress 安装 L2

📖 What — 这是什么

SSL 证书为网站启用 HTTPS 加密连接，将 http:// 升级为 https://。浏览器通过 SSL 证书验证网站身份并加密传输数据，防止中间人窃听或篡改。

没有 SSL 的网站在 Chrome 中会被标记为"不安全"，严重影响用户信任和 SEO 排名。

🎯 Why — 为什么要做

SSL 是现代网站的必备配置，Google 已将 HTTPS 作为排名因素。没有 SSL 的网站：浏览器显示警告、表单数据明文传输、搜索排名受损、Application Password 无法使用。

Hostinger 提供免费的 Let's Encrypt SSL 证书，没有理由不开启。

🔧 How — 怎么做

在 Hostinger 上配置 SSL

  1. 登录 Hostinger hPanel
  2. 选择目标网站
  3. 进入 Security → SSL
  4. 选择域名，点击 Install SSL（或 Setup）
  5. 等待证书签发（通常几分钟内完成）
  6. 开启 Force HTTPS（强制 HTTPS 跳转）

WordPress 端配置

  1. 登录 WordPress 后台
  2. 进入 设置 → 常规
  3. 将 WordPress 地址 (URL) 和 站点地址 (URL) 都改为 https:// 开头
  4. 保存更改
修改 URL 后会自动退出登录，用 https:// 版本重新登录即可。

⚠️ 常见问题与注意事项

  • SSL 证书签发需要域名 DNS 已生效并指向服务器，DNS 未生效时无法安装证书
  • Let's Encrypt 证书有效期 90 天，Hostinger 会自动续期，无需手动操作
  • 开启 HTTPS 后，如果页面出现"混合内容"警告（锁图标有警告），说明页面中有 http:// 资源，需要替换为 https://
  • 可以使用 Really Simple SSL 插件一键修复混合内容问题
  • 如果使用 Cloudflare，SSL 模式必须设为 Full (Strict)，否则可能出现重定向循环
  • 修改 WordPress URL 前确保 SSL 已经生效，否则会导致站点无法访问
