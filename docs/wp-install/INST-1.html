Hostinger 一键安装 WP

WordPress 安装 L1

📖 What — 这是什么

Hostinger 提供一键安装 WordPress 的功能，自动完成数据库创建、WordPress 文件部署、基础配置等所有步骤。整个过程只需填写站点标题和管理员信息，2 分钟内即可完成。

这是目前最简单的 WordPress 安装方式，适合绝大多数建站场景。

🎯 Why — 为什么要做

手动安装 WordPress 需要下载文件、创建数据库、配置 wp-config.php、上传文件等多个步骤。一键安装省去了所有技术细节，让教练可以快速搭建站点，把时间花在真正重要的配置上。

对于训练环境，一键安装是最高效的起步方式。

🔧 How — 怎么做

  1. 登录 Hostinger hPanel
  2. 点击 Websites → Add Website（或 Create or migrate a website）
  3. 选择 WordPress 作为建站平台
  4. 填写 WordPress 信息：
    • 网站标题（后续可修改）
    • 管理员邮箱
    • 管理员密码（建议使用强密码）
  5. 选择是否安装预设插件和主题（可跳过）
  6. 绑定域名或使用临时域名
  7. 选择服务器位置（数据中心）
  8. 点击安装，等待完成

安装完成后，通过 你的域名/wp-admin 即可访问 WordPress 后台。

⚠️ 常见问题与注意事项

  • 安装时设置的密码是 WordPress 管理员密码，不是 Hostinger 账号密码
  • 如果域名 DNS 尚未生效，可以先用 Hostinger 提供的临时域名访问
  • 安装后第一件事：修改固定链接为"文章名"结构
  • Hostinger 可能会预装一些推荐插件（如 Starter Templates），不需要的可以直接删除
  • 安装后建议立即检查 PHP 版本（hPanel → Advanced → PHP Configuration），确保使用 8.1+
