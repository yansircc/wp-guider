安装并激活插件

插件基础 L1

📖 What — 这是什么

插件（Plugin）是扩展 WordPress 功能的程序包。WordPress 核心提供基础功能，而插件可以添加联系表单、SEO 工具、安全防护、电商功能等几乎任何你能想到的功能。

WordPress 官方插件目录（wordpress.org/plugins）收录了数万个免费插件。安装插件是两步操作：先安装（下载文件到服务器），再激活（启用插件功能）。

🎯 Why — 为什么要做

WordPress 的强大之处在于其插件生态。通过安装插件，你可以在不写任何代码的情况下为网站添加复杂功能。例如用 WPForms 创建联系表单、用 Yoast SEO 优化搜索排名、用 WooCommerce 搭建在线商店。

掌握插件的安装和激活是 WordPress 运维的必备技能。

🔧 How — 怎么做

方法一：从 WordPress 插件目录安装

  1. 进入后台 插件 → 安装插件
  2. 在搜索框中输入插件名称（如「Yoast SEO」）
  3. 在搜索结果中找到目标插件，点击 现在安装
  4. 安装完成后，按钮变为 启用，点击启用插件

方法二：上传 ZIP 文件安装

  1. 进入 插件 → 安装插件
  2. 点击顶部 上传插件 按钮
  3. 选择插件的 .zip 文件，点击 现在安装
  4. 安装完成后点击 启用插件

此方法用于安装从第三方市场（如 CodeCanyon）购买的付费插件。

⚠️ 常见问题与注意事项

  • 安装 ≠ 激活：插件安装后默认是未激活状态，需要手动点击「启用」才会生效
  • 不要安装太多插件：每个插件都会增加服务器负载和潜在安全风险。建议控制在 15-20 个以内，只安装真正需要的
  • 检查插件质量：安装前查看「最后更新时间」「兼容 WordPress 版本」「活跃安装量」「评分」。超过一年未更新的插件要谨慎
  • 插件冲突：新安装的插件可能与现有插件或主题冲突，导致网站异常。安装后立即检查前台是否正常
  • 上传安装的安全风险：只从可信来源下载插件 ZIP 文件。来路不明的插件可能包含恶意代码
