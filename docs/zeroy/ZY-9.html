创建联系页（表单+联系方式）

ZeroY AI 建站L3

📖 What — 这是什么

联系页（Contact Us）是网站访客与企业建立联系的核心入口，通常包含在线询盘表单、公司地址、电话、邮箱等联系方式，有时还嵌入地图。ZeroY 可以用 AI 快速生成包含表单的联系页。

🎯 Why — 为什么要做

联系页是网站转化的关键环节。没有便捷的联系方式，潜在客户可能直接流失。在线表单降低了联系门槛，让访客无需打电话或发邮件就能提交询盘。

🔧 How — 怎么做

  1. WordPress 后台 → 页面 → 新增，标题 "Contact Us" 或 "联系我们" → 发布
  2. 用 ZeroY 编辑器打开
  3. 调用 AI 助手描述需求：
    • "创建一个联系页，左侧是询盘表单（姓名、邮箱、电话、公司名、消息），右侧是公司联系信息（地址、电话、邮箱、工作时间）"
  4. AI 生成后插入编辑器
  5. 替换联系信息为真实内容
  6. 确认表单字段满足业务需求
  7. 预览 → 保存

提示：表单提交的数据会进入 ZeroY 的表单提交记录，后续可配置邮件通知和感谢页跳转（参见 ZY.4 系列文档）。

⚠️ 常见问题与注意事项

  • 表单提交路径：ZeroY 表单必须使用 /wp-json/zeroy/v1/submissions 端点，AI 生成时通常已自动配置
  • 固定链接格式：表单提交需要"文章名"固定链接格式，否则 REST API 不可用（设置 → 固定链接 → 文章名）
  • 测试提交：创建完成后务必自己提交一次测试，确认数据能正常进入后台
  • 必填字段：建议至少将邮箱或电话设为必填，确保有回联方式
