查看表单提交记录 + 配置感谢页跳转

ZeroY AI 建站L2

本文同时覆盖: ZY-20, ZY-21

📖 What — 这是什么

ZeroY 内置表单系统会将访客提交的询盘数据保存在 WordPress 后台。你可以在"表单提交"面板中查看所有提交记录。同时，你可以配置感谢页 URL，让用户提交表单后自动跳转到感谢页面。

🎯 Why — 为什么要做

查看提交记录是了解客户询盘情况的基本操作。配置感谢页跳转则服务两个目的：一是给用户明确的"提交成功"反馈，二是为 Google Ads 等广告平台提供转化追踪的触发点。

🔧 How — 怎么做

查看表单提交记录

  1. 进入 WordPress 后台 → ZeroY 模块 → 表单提交（Form Submissions）
  2. 查看提交列表，包含提交时间、提交者信息和表单内容
  3. 点击某条记录查看详细信息

配置感谢页跳转

  1. 首先确保已创建好感谢页（参见 ZY.2-f），记下其 URL（如 /thank-you/）
  2. 进入 WordPress 后台 → ZeroY 模块 → 表单提交 → 设置
  3. 找到感谢页 URL（Thank You Page URL）输入框
  4. 填入感谢页的完整路径或 URL
  5. 保存设置
  6. 测试：在前台提交一次表单，确认自动跳转到感谢页

⚠️ 常见问题与注意事项

  • 提交后不跳转：检查固定链接格式是否设为"文章名"（设置 → 固定链接），REST API 需要该格式才能正常工作
  • URL 格式：感谢页 URL 可以用相对路径（如 /thank-you/）或完整 URL
  • 提交记录不丢失：即使未配置感谢页跳转，表单数据仍然会保存到后台
  • 定期检查：养成定期查看提交记录的习惯，及时回复客户询盘
