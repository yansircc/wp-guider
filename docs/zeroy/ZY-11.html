编辑器操作：插入/删除/撤回、前后台切换、素材管理

ZeroY AI 建站L2

本文同时覆盖: ZY-11, ZY-12, ZY-13

📖 What — 这是什么

ZeroY 编辑器除了 AI 生成能力，还有一套完整的手动操作体系：插入/删除/撤回元素、在前台预览与后台编辑之间切换、以及管理图片和图标等素材。这些是日常编辑的基础技能。

🎯 Why — 为什么要做

AI 生成的内容不可能一步到位。你需要手动删除多余区块、在指定位置插入新内容、替换占位图片为真实素材，并在前台确认实际效果。掌握这些操作才能把 AI 产出打磨成可交付的成品。

🔧 How — 怎么做

插入 / 删除 / 撤回

  • 插入：选中目标位置，用工具栏或 AI 助手在该位置插入新内容（文本、图片、按钮等）
  • 删除：选中元素或区块 → 按 Delete 键或点击工具栏删除按钮
  • 撤回：Ctrl+Z（Mac 为 Cmd+Z）撤回上一步；Ctrl+Shift+Z 重做
撤回记录仅存在于当前编辑会话，刷新页面会清空。重大修改前务必先保存。

前台预览与后台编辑切换

  1. 后台 → 编辑器：页面列表 → 点击「用 ZeroY 编辑」
  2. 编辑器 → 前台：保存后，新标签页打开页面 URL 查看实际效果
  3. 前台 → 编辑器：顶部管理工具栏点击「用 ZeroY 编辑」
用无痕窗口检查前台效果，可以排除管理员工具栏和缓存的影响。

添加 / 替换素材

  1. 替换图片：点击占位图 → 设置面板 → 从媒体库选择或上传新图片
  2. 添加图标：选中位置 → 用 AI 描述图标效果，或从内置图标库选择，也可上传 SVG
  3. 批量操作建议：先将所有素材上传到媒体库并统一命名（如 team-张三.jpg），再逐一替换

⚠️ 常见问题与注意事项

  • 未保存的修改前台看不到：每次编辑后先保存，再切换到前台预览
  • 删除操作要注意层级：删除整个区块会连带子元素一起移除，确认选中的是你想删的层级
  • 图片尺寸：过大影响加载速度，过小会模糊。一般 1200px 宽足够，推荐 WebP 格式
  • Alt 文本：替换图片时填写 Alt 文本，有助于 SEO 和无障碍访问
  • 前台缓存：如果看不到最新改动，用 Ctrl+Shift+R 强制刷新
