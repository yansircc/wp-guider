配置 AI：填写 API Key + API 端点 + 验证连接

ZeroY AI 建站L2

📖 What — 这是什么

ZeroY 的 AI 生成功能需要连接大语言模型服务。你需要在 ZeroY 设置中填写 AI 服务的 API Key 和 API 端点地址，让 ZeroY 编辑器能够调用 AI 来生成页面内容、修改布局等。常用的 AI 服务提供商包括 aihubmix.com 等。

🎯 Why — 为什么要做

API Token 只是连接 ZeroY 云端服务，AI 配置则是连接大语言模型。没有正确配置 AI 连接，你在编辑器中点击"AI 助手"会报错，无法使用自然语言生成页面内容这一核心功能。

🔧 How — 怎么做

第一步：获取 AI 服务凭证

  1. 访问 AI 服务提供商（如 aihubmix.com），注册并登录
  2. 在控制台找到 API Key（密钥）并复制
  3. 记下 API 端点地址（Endpoint URL）

第二步：在 ZeroY 中配置

  1. 进入 WordPress 后台 → ZeroY → AI Settings（AI 设置）
  2. 粘贴 API Key 到密钥输入框
  3. 填写 API 端点地址
  4. 点击验证连接按钮

第三步：确认连接成功

  1. 验证通过后，会显示连接成功提示
  2. 保存设置
  3. 进入任意页面的 ZeroY 编辑器，尝试调用 AI 助手确认功能正常

⚠️ 常见问题与注意事项

  • 端点地址不要写错：注意区分 v1/chat/completions 等路径，复制时确保完整
  • API Key 有余额限制：如果验证失败返回 403，很可能是额度用完了
  • 401 错误：说明 API Key 过期或无效，需要重新生成
  • 400 错误：通常是网络问题，检查服务器是否能访问外部 API 端点
  • 不要混淆 API Token 和 API Key：API Token 是 ZeroY 平台凭证，API Key 是 AI 服务凭证，两者用途不同
