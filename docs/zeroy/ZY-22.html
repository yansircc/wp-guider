配置邮件通知 + 安装 SMTP 插件

ZeroY AI 建站L3

本文同时覆盖: ZY-22, ZY-23

📖 What — 这是什么

ZeroY 支持在有新表单提交时自动发送邮件通知给指定收件人。默认使用 WordPress 内置的 wp_mail() 函数发送邮件，但这种方式在很多服务器上不稳定，需要安装 SMTP 插件来保证邮件可靠送达。

🎯 Why — 为什么要做

如果没有邮件通知，你只能登录 WordPress 后台才能看到新的询盘。配置邮件通知后，每当有人提交表单，你会立即收到邮件提醒，不会错过任何商机。SMTP 则确保邮件不会被丢弃或进入垃圾箱。

🔧 How — 怎么做

第一步：启用邮件通知

  1. 进入 WordPress 后台 → ZeroY → 设置（Settings）
  2. 找到邮件通知区域，启用 wp_mail
  3. 填写收件人邮箱（可以填多个，用逗号分隔）
  4. 设置邮件标题（如"新的网站询盘"）
  5. 保存设置

第二步：安装 SMTP 插件

  1. 进入 WordPress 后台 → 插件 → 安装插件
  2. 搜索 Fluent SMTP（推荐）或 WP Mail SMTP
  3. 安装并启用

第三步：配置 SMTP

  1. 进入 SMTP 插件设置页面
  2. 选择邮件服务商（如 Gmail、Outlook、阿里企业邮、自定义 SMTP 等）
  3. 填写 SMTP 服务器地址、端口、用户名和密码
  4. 设置发件人邮箱和名称
  5. 点击发送测试邮件验证配置
  6. 确认收到测试邮件后保存

第四步：验证完整流程

  1. 在前台提交一次表单
  2. 确认收到邮件通知
  3. 检查邮件内容是否包含表单数据

⚠️ 常见问题与注意事项

  • wp_mail 不可靠：这是最常见的坑——很多主机默认的 PHP mail() 函数被禁用或会进垃圾箱，强烈建议安装 SMTP 插件
  • Gmail SMTP 限制：Gmail 每天发送限额 500 封，且需要开启"应用专用密码"
  • 发件人地址：SMTP 发件人地址最好与域名一致（如 noreply@yourdomain.com），减少被标记为垃圾邮件的概率
  • 密码安全：SMTP 密码存储在数据库中，确保数据库安全
  • 测试邮件必做：配置完成后一定要发送测试邮件，不要假设配置正确
