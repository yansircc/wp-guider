Lottie 动画 Widget

Elementor 建站L3NEW

📖 What — 这是什么

Lottie 是一种轻量级矢量动画格式（基于 JSON），体积远小于 GIF/视频，且无限缩放不失真。Elementor Pro 内置 Lottie Widget，可以直接在页面中嵌入 Lottie 动画文件，用于 Hero 区域、加载动画、图标动效等场景。

🎯 Why — 为什么要做

  • 视觉吸引力：动画比静态图片更能抓住用户注意力
  • 性能友好：一个 Lottie JSON 通常只有几十 KB，比同等效果的 GIF 小 10 倍以上
  • 设计质感：Lottie 动画通常由专业设计师制作，能显著提升页面品质感

🔧 How — 怎么做

第一步：获取 Lottie 动画文件

  1. 访问 LottieFiles.com（全球最大的 Lottie 动画库）
  2. 搜索你需要的动画效果（如 "loading"、"success"、"rocket"）
  3. 下载 JSON 格式文件，或复制动画的 URL

第二步：在 Elementor 中使用

  1. 编辑页面 → 从左侧搜索 "Lottie" Widget → 拖到画布
  2. 在 Content Tab 选择来源：
    • Media File：上传 JSON 文件到媒体库
    • External URL：粘贴 LottieFiles 的动画 URL
  3. 配置播放行为：
    • Autoplay：页面加载后自动播放
    • Loop：循环播放
    • On Hover：鼠标悬停时播放
    • Scroll：滚动到可视区域时播放
  4. 在 Style Tab 调整尺寸和对齐方式

⚠️ 常见问题与注意事项

  • 不要过度使用：一个页面 1-3 个 Lottie 动画即可，过多动画会分散注意力且影响性能
  • 版权注意：LottieFiles 上部分动画有版权限制，注意查看许可协议
  • Fallback：极少数旧浏览器可能不支持 Lottie，但现代浏览器都兼容
  • 文件大小：虽然 Lottie 通常很小，但包含大量路径的复杂动画也可能达到几百 KB，上传前检查文件大小
