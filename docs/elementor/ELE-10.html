全局样式与设计系统

Elementor 建站L2-L3

本文同时覆盖: ELE-10, ELE-11, ELE-12

📖 What — 这是什么

Elementor 的设计系统包含三个层次：全局颜色/字体（基础）、Design System Variables（进阶变量）和 CSS Classes（样式复用）。它们共同构成一个中心化的设计管理体系，让你一处修改、全站生效。

🎯 Why — 为什么要做

  • 一致性：统一的颜色和字体确保全站视觉一致，不会出现"每个页面看起来像不同网站"的问题
  • 效率：改一次全局颜色，所有引用该颜色的地方自动更新，不用逐页修改
  • 可维护：客户要换品牌色？改一个变量就行，而不是翻遍几十个页面
  • 团队协作：统一的设计规范让多人协作时保持一致

🔧 How — 怎么做

全局颜色（L2）

  1. 在编辑器左上角汉堡菜单 → Site Settings → Global Colors
  2. 设置四个核心颜色：Primary（主色）、Secondary（辅助色）、Text（正文色）、Accent（强调色）
  3. 可以添加自定义颜色组
  4. 使用时：在任何颜色选择器中点击"地球"图标，选择全局颜色

全局字体（L2）

  1. Site Settings → Global Fonts
  2. 设置四个字体层级：Primary（标题）、Secondary（副标题）、Text（正文）、Accent（强调）
  3. 每个层级包含：字体族、大小、粗细、行高、字间距
  4. 使用时：在排版设置中点击"地球"图标选择全局字体

Design System Variables（L3，3.31+）

在全局颜色/字体之上，Elementor 3.31 引入了更精细的变量系统：

  • 进入 Site Settings → Design System
  • 可定义变量类型：字号（font-size）、间距（spacing）、圆角（border-radius）等
  • 例如：定义 --spacing-sm: 8px、--spacing-md: 16px、--spacing-lg: 32px
  • 在 Widget 的间距、圆角等属性中引用这些变量

CSS Classes（L3，3.29+）

  1. 进入 Site Settings → CSS Classes
  2. 创建一个 Class（如 card-shadow），定义样式规则
  3. 在任何 Widget 的 Advanced Tab → CSS Classes 中应用该 Class
  4. 修改 Class 定义后，所有使用该 Class 的元素自动更新

⚠️ 常见问题与注意事项

  • 先设后用：在开始设计页面之前先设好全局颜色和字体，避免后期大量返工
  • 命名规范：全局颜色/变量使用语义化命名（如 "Primary"、"CTA Background"），不要用 "Red"、"#FF0000" 这类具体值命名
  • 版本要求：Design System Variables 需要 Elementor 3.31+，CSS Classes 需要 3.29+。确认版本后再使用
  • 不要过度：初学阶段先用好全局颜色和字体即可，Variables 和 CSS Classes 在站点规模较大时才有明显收益
