Portfolio Widget 作品集展示

Elementor 建站L4

📖 What — 这是什么

Portfolio Widget 是 Elementor Pro 的作品集展示组件，以网格形式展示文章或自定义 Post Type 的特色图，并提供分类标签筛选（Filter Bar）功能。用户可以点击标签快速切换显示不同分类的内容，无需刷新页面。

它本质上是 Posts Widget 的变体，专门针对视觉展示场景优化 — 强调图片，弱化文字。

🎯 Why — 为什么要做

  • 视觉展示：设计公司、摄影师、建筑事务所等需要以图片为主的作品展示方式
  • 即时筛选：Filter Bar 让用户无需跳转页面即可按分类浏览不同类型的作品
  • 响应式网格：自动适配不同屏幕尺寸，保持视觉效果

🔧 How — 怎么做

  1. 拖入 Portfolio Widget 到画布
  2. 在 Query 中设置数据源（通常是文章或自定义 Post Type 如"作品"）
  3. 配置显示选项：
    • Columns：列数（通常 3 或 4 列）
    • Posts Per Page：显示数量
    • Image Ratio：图片宽高比（如 1:1 正方形、3:2 横向、4:3 等）
    • Item Gap：项目间距
  4. 开启 Filter Bar：
    • Taxonomy：选择用哪个分类法做筛选标签（如 Category、Tag 或自定义 Taxonomy）
    • Filter Bar 会自动生成当前内容涉及的所有分类标签
  5. 在 Style Tab 中设置悬停效果（Overlay）、标题样式、Filter Bar 样式等

悬停效果（Overlay）

Portfolio Widget 支持鼠标悬停时显示 Overlay 层，可以包含文章标题和分类信息。在 Style → Overlay 中设置背景色、透明度和动画效果。

⚠️ 常见问题与注意事项

  • Portfolio Widget 要求所有文章都有 特色图，没有特色图的文章会显示空白占位
  • Filter Bar 的筛选是前端 JavaScript 实现的（隐藏/显示），不是 AJAX 请求，所以所有文章其实已经加载了 — 文章过多会影响性能
  • 如果作品数量超过 30 个，考虑分页或使用 Posts Widget + 自定义筛选方案替代
  • Portfolio Widget 在 Elementor 较新版本中可能被标记为"Legacy"，官方推荐使用 Loop Grid + Loop Carousel 作为替代方案
  • 特色图的尺寸和比例尽量统一，否则网格会参差不齐
