Reusable Sections 可复用区块

Elementor 建站L3NEW

📖 What — 这是什么

可复用区块是将常用的页面区块（如 CTA 横幅、FAQ 区块、联系信息区）保存为独立模板，然后通过 Template Widget 在多个页面中引用。关键区别：引用的是同一个模板实例，修改一次所有引用处同步更新。

🎯 Why — 为什么要做

  • 维护一处，全站同步：CTA 区块出现在 10 个页面中，修改一次模板 10 个页面同时更新
  • 保持一致性：避免手动复制导致不同页面的同一区块出现细微差异
  • 效率提升：新页面直接插入已有模板，不必从零搭建重复区块

🔧 How — 怎么做

创建可复用区块

  1. 在 Elementor 编辑器中设计好目标区块
  2. 右键点击区块的 Container → Save as Template
  3. 命名（如 "CTA-底部横幅"）→ 保存

在其他页面引用

  1. 编辑目标页面
  2. 从左侧搜索 Template Widget（或 "Saved Template"）
  3. 拖到需要的位置
  4. 在 Content Tab 的下拉菜单中选择之前保存的模板

修改模板（全局同步）

  1. WordPress 后台 → 模板 → Saved Templates
  2. 找到目标模板 → 用 Elementor 编辑
  3. 修改内容 → 更新
  4. 所有引用该模板的页面自动更新（可能需要清除缓存才能看到变化）

⚠️ 常见问题与注意事项

  • Template Widget vs 复制粘贴：复制粘贴创建的是独立副本，修改不会同步。Template Widget 是引用，全局同步。根据需求选择
  • 嵌套限制：不要在模板中再嵌套 Template Widget，会导致递归加载问题
  • 样式隔离：模板中的样式可能和宿主页面产生冲突（如字号、边距），插入后检查一下
  • 缓存影响：修改模板后如果前台未更新，清除 LiteSpeed/Elementor 缓存
