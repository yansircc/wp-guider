Elementor 表单

Elementor 建站L2-L3

本文同时覆盖: ELE-28, ELE-29, ELE-30, ELE-31

📖 What — 这是什么

Elementor Pro 内置强大的表单构建器，无需额外表单插件。支持拖拽式字段配置、多步骤表单、提交后自动化操作（邮件通知、Webhook、CRM 集成等）。这是 Elementor Pro 最有价值的功能之一。

🎯 Why — 为什么要做

  • 线索收集：联系表单、询盘表单是外贸/服务类网站的核心转化工具
  • 减少插件：不需要额外安装 Contact Form 7、WPForms 等插件
  • 样式统一：表单直接在 Elementor 中设计，与页面风格完全一致
  • 自动化：提交后自动发邮件、推送到 CRM，减少人工处理

🔧 How — 怎么做

基础联系表单（L2）

  1. 拖入 Form Widget
  2. 默认字段：Name、Email、Message
  3. 添加/删除字段，可用字段类型：
    • Text、Email、Textarea、Tel（电话）
    • Select（下拉）、Radio（单选）、Checkbox（多选）
    • Number、Date、Time、Upload（文件上传）
    • Hidden（隐藏字段，用于追踪来源）
  4. 每个字段设置：Label（标签）、Placeholder（占位文字）、Required（必填）、Width（宽度比例）
  5. Submit Button：自定义按钮文字、样式和加载动画

多步骤表单（L3）

  1. 在表单中添加 Step 字段，将表单分为多个步骤
  2. 每个 Step 之间的字段属于同一步
  3. 自动生成进度指示器（步骤条）
  4. 设置 Step 类型：Number（数字步骤）、Text（文字步骤）、Icon（图标步骤）
  5. 典型用途：报价申请（第一步基本信息 → 第二步需求详情 → 第三步确认提交）

提交后动作 Actions After Submit（L3）

在表单的 Actions After Submit 区域添加动作：

  • Email：发送通知邮件到指定邮箱（可自定义模板、使用字段变量）
  • Email 2：发送自动回复邮件给提交者
  • Redirect：提交后跳转到感谢页（用于追踪转化）
  • Webhook：将表单数据 POST 到自定义 URL（接入任何系统）
  • Popup：提交后关闭/打开 Popup

CRM 集成（L3）

  • Mailchimp：在 Elementor → Settings → Integrations 中输入 Mailchimp API Key
  • 在 Actions 中添加 Mailchimp，选择 List 和字段映射
  • HubSpot：类似配置，将表单数据推送到 HubSpot CRM
  • 其他集成：ActiveCampaign、ConvertKit、Drip、GetResponse 等

⚠️ 常见问题与注意事项

  • 邮件不送达：WordPress 默认使用 PHP mail() 发送邮件，容易进垃圾箱。务必安装 SMTP 插件（如 WP Mail SMTP）配置第三方邮件服务
  • 防垃圾：在表单设置中启用 reCAPTCHA 或 Honeypot，防止垃圾提交
  • 提交记录：Elementor Pro 会将表单提交保存到 Elementor → Submissions，即使邮件发送失败也不会丢失数据
  • GDPR：如果面向欧洲用户，需要添加隐私政策同意复选框
  • 多步骤体验：步骤不宜超过 3-4 步，每步字段控制在 3-5 个，减少用户放弃率
