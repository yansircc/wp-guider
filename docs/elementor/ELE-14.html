按设备隐藏/显示元素

Elementor 建站L3NEW

📖 What — 这是什么

Elementor 允许你按设备类型（桌面/平板/手机）控制元素是否显示。比如桌面端显示大横幅图，手机端隐藏；或者只在手机端显示「一键拨号」按钮。

🎯 Why — 为什么要做

  • 不同设备的屏幕空间不同，某些元素在小屏幕上没有意义或影响体验
  • 移动端可能需要专属的交互元素（如浮动电话按钮、底部导航栏）
  • 简化移动端布局，提升加载速度和用户体验

🔧 How — 怎么做

  1. 选中目标 Widget（或 Container/Section）
  2. 进入 Advanced 标签 → Responsive
  3. 找到「Visibility」（可见性）区域
  4. 按需切换各设备的显示状态：
    • Hide on Desktop：桌面端隐藏
    • Hide on Tablet：平板隐藏
    • Hide on Mobile：手机端隐藏

编辑器中的视觉提示

被隐藏的元素在编辑器中会显示为半透明并带有一个紫色的隐藏图标，表示该元素在当前预览设备下不可见。切换底部工具栏的设备预览模式可以验证效果。

实战示例

  • 桌面大图 + 手机小图：创建两个 Image Widget，一个放大图（Hide on Mobile），一个放小图（Hide on Desktop + Hide on Tablet）
  • 手机专属 CTA：创建一个浮动按钮 Container，设置 Hide on Desktop
底部工具栏的响应式切换按钮（桌面/平板/手机图标）可以快速预览不同设备下的效果。

⚠️ 常见问题与注意事项

  • 隐藏 ≠ 不加载：Elementor 的响应式隐藏是通过 CSS display: none 实现的，HTML 仍然会加载。对性能有极致要求时需注意
  • Container 级别隐藏：如果整个区域在某个设备不显示，在 Container/Section 级别隐藏比逐个 Widget 隐藏更高效
  • 断点自定义：默认断点是 1025px（桌面/平板）和 768px（平板/手机）。可在汉堡菜单 → Site Settings → Layout → Breakpoints中自定义
