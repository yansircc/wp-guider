按钮 / 图标 / 间距 Widget

Elementor 建站L1

📖 What — 这是什么

这三个是 Elementor 最常用的辅助 Widget：

  • Button（按钮）：行动号召按钮，支持链接、图标、Hover 效果
  • Icon（图标）：展示 Font Awesome 或自定义 SVG 图标
  • Spacer（间距）：在元素之间添加可控的垂直空间

🎯 Why — 为什么要做

按钮是转化的核心元素——「立即咨询」「免费试用」「了解更多」——每个页面都需要。图标用于增强视觉传达（服务特性、联系方式、社交媒体）。间距控制页面呼吸感，避免元素过于紧凑。

🔧 How — 怎么做

Button 按钮

  1. 从 Widget 面板拖入 Button
  2. Content Tab：设置按钮文字、链接 URL、图标（可选）、对齐方式
  3. Style Tab：设置背景色、文字色、边框、圆角、内边距
  4. Hover 样式：Style Tab 中切换到 Hover 状态，设置悬停时的颜色变化和动画效果
按钮 ID 可在 Advanced Tab → CSS ID 中设置，用于页内锚点跳转（链接填 #id）。

Icon 图标

  1. 拖入 Icon Widget
  2. 点击图标区域，从图标库中选择（支持 Font Awesome 全套图标）
  3. Style Tab 设置颜色、大小、旋转角度
  4. 可选择 View 模式：Default（纯图标）/ Stacked（带背景）/ Framed（带边框）

Spacer 间距

  1. 拖入 Spacer Widget 到两个元素之间
  2. 拖拽控制柄或输入数值设置高度（px）
  3. 可为不同设备设置不同间距（点击设备图标切换）

⚠️ 常见问题与注意事项

  • 按钮链接默认在当前页打开，外部链接建议勾选「在新窗口打开」
  • Spacer Widget 可以用 Container 的 Padding/Margin 替代——优先使用 Container 间距，Spacer 仅在需要可视化控制时使用
  • 图标颜色建议使用全局颜色变量，方便后期统一调整
  • 按钮文字要简洁有力——「立即咨询」优于「如果您有兴趣请点击此处联系我们」
