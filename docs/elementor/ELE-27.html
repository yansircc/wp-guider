Sticky 元素（固定导航 / 侧边栏）

Elementor 建站L3NEW

📖 What — 这是什么

Sticky（粘性定位）让某个元素在页面滚动时固定在屏幕的指定位置。最常见的用法是固定导航栏（用户向下滚动时 Header 始终可见）和固定侧边栏（如目录、CTA 按钮）。Elementor Pro 的 Motion Effects 提供了内置的 Sticky 选项。

🎯 Why — 为什么要做

  • 导航始终可达：固定 Header 让用户在任何位置都能快速导航，不必滚回顶部
  • 提升转化：固定 CTA 按钮或联系方式在视口中，增加用户触达概率
  • 阅读辅助：长文章页固定目录（TOC）方便读者跳转章节

🔧 How — 怎么做

固定导航栏（Header Sticky）

  1. 用 Theme Builder 编辑 Header 模板
  2. 选中 Header 最外层的 Container
  3. 左侧面板 → Advanced Tab → Motion Effects
  4. 找到 Sticky 选项，设为 Top
  5. 可选：设置 Offset（距顶部偏移量）和 Effects Offset（触发距离）

固定侧边栏 / CTA

  1. 选中侧边栏 Container 或 CTA Widget
  2. Advanced → Motion Effects → Sticky → Top
  3. 设置 Stay in Column：打开此选项让元素只在其所在列内固定，滚出列范围后自动脱离
Sticky 支持按设备类型开关：可以在桌面端开启固定导航，但在手机端关闭（因为手机屏幕小，固定 Header 占空间）。

⚠️ 常见问题与注意事项

  • z-index 冲突：Sticky 元素固定后可能被其他元素遮挡，需要在 Advanced → Z-Index 中设置较高的值（如 999）
  • 背景透明问题：Header 固定后如果背景透明，文字会和页面内容重叠。务必给 Sticky Header 设置不透明背景色
  • 高度跳动：Header 变为 Sticky 后脱离文档流，页面内容可能突然上移。Elementor 通常自动补偿，但需要预览确认
  • 移动端测试：固定 Header 在小屏设备上占据大量空间，建议手机端使用更紧凑的 Sticky Header 或关闭
