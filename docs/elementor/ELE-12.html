CSS Classes 复用样式

Elementor 建站L3NEW

📖 What — 这是什么

通过为 Elementor Widget 添加自定义 CSS Class，配合全局样式表，可以实现样式的跨页面复用。比如定义一个 .card-shadow 类，任何 Widget 加上这个 class 就自动获得统一的阴影效果。

🎯 Why — 为什么要做

  • 一处修改，全站生效：改 CSS 类的定义 = 所有使用该类的元素同步更新
  • 减少重复工作：不需要逐个 Widget 调整样式设置
  • 弥补 Elementor 的限制：有些样式效果无法通过面板实现（如复杂 hover 效果、伪元素），CSS Class 是唯一方案

🔧 How — 怎么做

Step 1：定义全局 CSS

进入 Elementor 编辑器 → 左上角汉堡菜单 → Site Settings → Custom CSS，添加样式：

/* 卡片阴影 */
.card-shadow {
  box-shadow: 0 4px 20px rgba(0,0,0,0.08);
  border-radius: 12px;
  transition: box-shadow 0.3s;
}
.card-shadow:hover {
  box-shadow: 0 8px 30px rgba(0,0,0,0.15);
}

/* 文字渐变 */
.text-gradient {
  background: linear-gradient(135deg, #667eea, #764ba2);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
}

/* 统一间距 */
.section-padding {
  padding: 80px 0;
}

Step 2：为 Widget 添加 CSS Class

  1. 选中目标 Widget
  2. 进入 Advanced 标签 → Layout → CSS Classes
  3. 输入 class 名称（如 card-shadow），多个 class 用空格分隔

Step 3：验证

保存后在前台查看效果。如果不生效，检查 CSS 选择器优先级（可能需要加 !important 或更具体的选择器）。

建议 CSS Class 命名用 .ele-xxx 前缀（如 .ele-card、.ele-btn-primary），避免与主题/插件的 class 冲突。

⚠️ 常见问题与注意事项

  • Site Settings CSS vs Widget CSS：Site Settings 的 Custom CSS 是全局的；Widget 的 Custom CSS 只作用于该 Widget。复用样式应放在全局
  • Custom CSS 是 Pro 功能：Widget 级别的 Custom CSS 需要 Elementor Pro；Site Settings 级别的也需要 Pro
  • 替代方案：如果没有 Pro，可以在外观 → 自定义 → 额外 CSS（WordPress 自带功能）中添加全局样式
  • 不要滥用 !important：优先通过更具体的选择器（如 .elementor-widget.card-shadow）解决优先级问题
