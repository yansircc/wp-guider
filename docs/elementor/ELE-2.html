创建第一个页面（编辑器界面认识）

Elementor 建站L1

📖 What — 这是什么

Elementor 编辑器是一个前端可视化页面构建器，左侧是 Widget 面板和设置区，右侧是页面实时画布。所有操作都是拖拽式的，所见即所得。

创建第一个页面是熟悉编辑器的最佳方式——了解 Widget 面板、Container 结构、三个设置 Tab（Content / Style / Advanced）以及发布流程。

🎯 Why — 为什么要做

Elementor 的编辑器界面与 WordPress 原生编辑器（Gutenberg）完全不同。不熟悉界面布局会导致操作效率低下，甚至找不到需要的功能。

理解编辑器的核心概念——Container（容器）、Widget（组件）、Navigator（导航器）——是后续所有 Elementor 操作的基础。

🔧 How — 怎么做

1. 创建新页面并进入编辑器

  1. 后台 → 页面 → 新建页面
  2. 输入标题（如「测试页面」）
  3. 点击蓝色按钮 「使用 Elementor 编辑」

2. 认识编辑器界面

  • 左侧面板：Widget 库（可搜索）、页面设置、全局设置
  • 中间画布：页面实时预览，点击任意元素即可编辑
  • 底部工具栏：响应式预览切换、撤销/重做、保存/发布
  • Navigator（左下角图标）：页面结构树状视图，方便选择嵌套元素

3. 添加第一个内容

  1. 点击画布中的 「+」 按钮添加 Container
  2. 从左侧面板拖一个 Heading Widget 到 Container 中
  3. 在左侧 Content Tab 中修改标题文字
  4. 切换到 Style Tab 修改颜色、字号
  5. 切换到 Advanced Tab 设置边距（Margin/Padding）

4. 发布页面

点击左下角绿色 「发布」 按钮，确认后页面即生效。

右键点击画布中的元素可以复制、粘贴样式、复制整个 Widget，这是提高效率的重要操作。

⚠️ 常见问题与注意事项

  • 页面模板建议选择 Elementor Full Width（无侧边栏）或 Elementor Canvas（无 Header/Footer）
  • 编辑过程中 Elementor 会自动保存草稿，但建议定期手动点击「更新」
  • 如果左侧面板没有显示 Widget，点击左上角的网格图标回到 Widget 面板
  • Navigator（结构导航器）是处理复杂页面的利器——层级多时用它选元素比在画布上点击精确得多
