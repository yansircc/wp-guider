Theme Builder Footer

Elementor 建站L3

📖 What — 这是什么

Elementor Theme Builder 的 Footer 模板功能，允许你用可视化拖拽方式设计全站页脚，替代主题默认的 Footer。你可以在 Footer 中放置公司信息、导航链接、社交图标、版权声明等任意内容。

Footer 模板创建后会自动应用到全站所有页面，也可以通过 Display Conditions 指定只在特定页面显示。

🎯 Why — 为什么要做

  • 品牌一致性：统一全站页脚设计，确保每个页面底部信息一致
  • 灵活布局：主题默认 Footer 通常只能改文字和颜色，Theme Builder 可以做任意布局
  • 多 Footer 方案：不同页面可以使用不同的 Footer（如首页用大 Footer，内页用精简版）

🔧 How — 怎么做

  1. 进入 Templates → Theme Builder
  2. 点击 Footer 区域的 "+" 按钮（或 Add New → Footer）
  3. 选择一个预设模板或从空白开始
  4. 设计 Footer 内容，常见元素：
    • Container：创建多列布局（通常 3-4 列）
    • Heading / Text Editor：公司名称、简介、版权声明
    • Nav Menu：页脚导航链接（关于我们、隐私政策等）
    • Social Icons：社交媒体链接图标
    • Site Logo：品牌 Logo
  5. 设置 Footer 背景色（通常深色）和内边距
  6. 点击 Publish，在弹出的 Display Conditions 中选择 Entire Site

版权声明动态年份

在 Text Editor 中使用 Elementor 的 Dynamic Tags 插入当前年份，避免每年手动改：

点击文本框中的 Dynamic 图标 → 选择 Current Date Time → 格式设为 Y（只显示年份）。

⚠️ 常见问题与注意事项

  • 创建 Footer 模板后，主题原有的 Footer 会被完全替代
  • 如果页面底部出现两个 Footer，检查是否同时有主题 Footer 和 Elementor Footer 在生效
  • Footer 中的导航链接建议使用 Nav Menu Widget 而非手动写链接，方便后期维护
  • 移动端记得检查 Footer 的响应式布局，多列布局在手机上通常需要改为单列
  • Footer 设计保持简洁 — 信息过多的 Footer 反而降低用户体验
