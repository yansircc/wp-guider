动效、Lottie 与 Sticky

Elementor 建站L3

本文同时覆盖: ELE-25, ELE-26, ELE-27

📖 What — 这是什么

Motion Effects 为元素添加滚动触发的动画效果（视差、淡入、缩放等）。Lottie 是一种轻量级的 JSON 动画格式，可以替代 GIF 实现更流畅的矢量动画。Sticky 让元素在滚动时固定在屏幕上（如固定导航栏、侧边栏 CTA）。

🎯 Why — 为什么要做

  • 视觉吸引力：适度的动效让页面更有活力，引导用户注意力
  • 品牌质感：Lottie 动画可以展现品牌个性（加载动画、空状态、微交互）
  • 实用功能：Sticky 导航提升长页面的导航体验；Sticky CTA 提升转化

🔧 How — 怎么做

Motion Effects 动效

  1. 选中任意 Widget 或 Container
  2. 进入 Advanced Tab → Motion Effects
  3. Entrance Animation（入场动画）：
    • Fade In / Fade In Up / Fade In Left 等
    • 设置动画持续时间和延迟
  4. Scrolling Effects（滚动效果，Pro）：
    • Vertical Scroll：视差效果（元素滚动速度与页面不同）
    • Horizontal Scroll：水平位移
    • Transparency：滚动时渐变透明度
    • Blur / Rotate / Scale：滚动触发的模糊/旋转/缩放
  5. Mouse Effects（Pro）：元素跟随鼠标移动或产生 3D 倾斜效果

Lottie 动画

  1. 到 LottieFiles 下载或制作 JSON 动画文件
  2. 拖入 Lottie Widget（Pro 功能）
  3. 上传 JSON 文件或输入 LottieFiles URL
  4. 设置触发方式：
    • Auto Play：自动循环播放
    • On Scroll：根据滚动进度播放（滚动到 50% 时播放到 50%）
    • On Hover / On Click：鼠标交互触发
  5. 调整播放速度和循环次数

Sticky 元素

  1. 选中要固定的 Container 或 Widget
  2. Advanced Tab → Motion Effects → Sticky
  3. 选择固定位置：Top（固定在顶部）或 Bottom
  4. 设置 Offset（距离顶部/底部的偏移量）
  5. 设置 Stay in Column：开启后元素只在其父 Container 范围内固定，滚出父容器后取消固定
  6. 设置 Effects Offset：滚动多少距离后才触发固定

⚠️ 常见问题与注意事项

  • 动效适度：页面上有 1-3 个关键动效即可，满屏都在动会分散注意力、降低专业感
  • 性能影响：Scrolling Effects 和 Mouse Effects 会增加渲染负担，手机端建议关闭复杂动效
  • Lottie 大小：单个 Lottie JSON 文件建议控制在 100KB 以内
  • Sticky 层级：Sticky 元素需要设置较高的 Z-Index（如 100），避免被其他内容遮挡
  • Sticky + Header：如果页面有 Sticky Header，下方的 Sticky 元素需要增加 Offset 值，避免重叠
  • 可及性：有些用户对动画敏感（前庭障碍），应尊重系统的 prefers-reduced-motion 设置
