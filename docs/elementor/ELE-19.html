Animated Headline 动态标题

Elementor 建站L2NEW

📖 What — 这是什么

Animated Headline 是 Elementor Pro 的动态标题 Widget，可以让标题中的部分文字以动画方式切换。例如：「我们提供 [网站设计 / SEO 优化 / 品牌策划] 服务」，方括号中的文字会自动轮播切换。

支持两种动画风格：Highlighted（高亮标注，如下划线、圆圈、删除线等装饰效果）和 Rotating（旋转切换，文字逐个淡入淡出）。

🎯 Why — 为什么要做

动态标题可以在一个标题位置展示多条信息，吸引用户注意力。常用于 Hero Section 传达多个价值主张，或在服务页面轮播展示不同服务类型。

适度的动画效果增加页面活力，但要避免过度使用导致注意力分散。

🔧 How — 怎么做

Highlighted 高亮标注模式

  1. 拖入 Animated Headline Widget
  2. Style 选择 Highlighted
  3. 输入 Before Text（前缀文字）和 Highlighted Text（高亮部分）
  4. 选择 Shape（装饰形状）：Circle、Underline、Double Underline、Strikethrough 等
  5. Style Tab 设置高亮颜色和动画时长

Rotating 旋转切换模式

  1. Style 选择 Rotating
  2. 输入 Before Text（固定前缀）
  3. 在 Rotating Text 中每行输入一个切换词
  4. 选择动画效果：Typing（打字机）、Clip、Flip、Slide、Drop 等
Typing（打字机效果）是最自然、最不突兀的动画——适合大多数商业站点。Flip 和 Slide 更适合年轻活力的品牌。

⚠️ 常见问题与注意事项

  • Rotating 文字长度尽量相近，避免切换时布局跳动
  • 一个页面最多一处 Animated Headline——多处会让页面显得杂乱
  • 动画速度不宜过快，用户需要有时间阅读每条文字（建议 2-3 秒/条）
  • 确保动画文字在不播放时也能传达完整含义（考虑 SEO 和无障碍访问）
