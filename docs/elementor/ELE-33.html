Popup Builder 弹窗

Elementor 建站L3-L4

本文同时覆盖: ELE-33, ELE-34, ELE-35

📖 What — 这是什么

Elementor Pro 的 Popup Builder 允许你用与页面编辑相同的拖拽方式创建弹窗。支持三种类型：Modal（居中模态框）、Slide-in（侧滑进入）、Bar（顶部/底部通知条）。每个 Popup 可以配置精确的触发条件和显示规则。

🎯 Why — 为什么要做

  • 线索收集：弹出订阅表单、询盘表单的转化率通常高于页面内嵌表单
  • 促销通知：限时优惠、新品发布等信息通过弹窗即时传达
  • 挽留用户：Exit Intent（退出意图）弹窗在用户要离开时做最后挽留
  • 无需插件：不用安装额外的 Popup 插件，样式与网站完全统一

🔧 How — 怎么做

创建 Popup（L3）

  1. 进入 Templates → Popups → 添加新 Popup
  2. 选择预设模板或空白开始
  3. 设置 Popup Layout：
    • Width / Height：弹窗尺寸
    • Position：Center / Top / Bottom / Left / Right
    • Overlay：是否显示背景遮罩（点击遮罩关闭）
    • Close Button：关闭按钮样式和位置
    • Entrance / Exit Animation：进入/退出动画
  4. 像编辑普通页面一样添加内容（标题、文本、表单、图片等）

触发条件 Triggers（L4）

在 Popup 的 Publishing Settings → Triggers 中设置：

  • On Page Load：页面加载后 X 秒弹出
  • On Scroll：滚动到页面 X% 时弹出
  • On Scroll to Element：滚动到特定元素时弹出
  • On Click：点击指定元素时弹出（需要在触发元素上设置 CSS ID 或 Class）
  • After X Page Views：用户访问了 X 个页面后弹出
  • On Exit Intent：检测到鼠标移向浏览器关闭按钮时弹出（桌面端）
  • After Inactivity：用户 X 秒无操作后弹出

显示规则 Display Conditions（L4）

  • Include：在哪些页面显示（全站 / 指定页面 / 分类 / 文章类型）
  • Exclude：在哪些页面不显示
  • Advanced Rules：
    • Show to Logged In / Logged Out 用户
    • Show on specific Devices（桌面/平板/手机）
    • Show based on Referral Source（来自特定来源的用户）
    • Show X times per session / 不再重复显示

Popup 表单集成（L4）

  1. 在 Popup 中放入 Form Widget
  2. 配置 Actions After Submit（邮件/Webhook/Mailchimp 等）
  3. 设置提交后动作：关闭 Popup / 跳转感谢页
  4. 典型场景：Exit Intent 弹窗 + 订阅表单 + Mailchimp 集成

⚠️ 常见问题与注意事项

  • 不要滥用：弹窗过于频繁会严重损害用户体验。一个页面最多触发一个弹窗
  • Google 处罚：Google 会对移动端的"侵入式插页广告"（Intrusive Interstitials）降权。避免在移动端使用全屏 Popup
  • 关闭方式：必须提供明显的关闭按钮，以及点击 Overlay 关闭的选项
  • 频率控制：使用 "Don't Show Again" 规则，避免同一用户反复看到同一弹窗
  • Exit Intent 限制：Exit Intent 在移动端不可用（没有鼠标光标），需要用其他触发方式替代
  • On Click 触发：在页面中的按钮需要添加 Dynamic Tag Link → Popup → 选择对应 Popup，或者使用 CSS Class elementor-popup-trigger
